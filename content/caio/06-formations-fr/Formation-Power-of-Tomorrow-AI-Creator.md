### MODULE 3 — Prompt Engineering Fondamental — L'Art de Communiquer avec l'IA

**Objectif** : L'eleve maitrise les bases du prompting pour TOUS les types de modèles.

**Lecons a générer :**

3.1 **Les principes universels du prompting**
- Specificite : plus tu es précis, meilleur est le resultat
- Structure : organiser ses instructions (contexte, tache, format, contraintes)
- Iteration : le premier essai est un brouillon, pas le resultat final
- Exemples : montrer ce qu'on veut (few-shot learning)
- Negation : dire ce qu'on ne veut PAS

3.2 **Prompt engineering pour le texte**
- Framework RACE : Role + Action + Context + Expectation
- Chain-of-thought : "Reflechis étape par étape"
- Few-shot : fournir 2-3 exemples du format voulu
- Templates reutilisables par type de contenu
- Exercice : ecrire 5 prompts pour 5 types de contenu (blog, email, script, social, newsletter)

3.3 **Prompt engineering pour les images**
- Structure : Sujet + Environnement + Style + Eclairage + Camera
- Prompts negatifs : `--no hands, text, watermark, blurry`
- Poids de prompt : `car::2 jungle::1` (Midjourney)
- Références de style : `--sref` (Midjourney), style descriptions (DALL-E)
- Seed control : reproduire et varier un resultat
- Exercice : générer la même scene dans 5 styles différents

3.4 **Prompt engineering pour la video**
- Decrire les FORCES, pas les apparences ("le vent souffle les feuilles" > "feuilles qui bougent")
- Keywords camera : dolly, tracking, crane, pan, tilt, orbit, zoom
- Specifier 24fps pour un look cinematique
- Eclairage et couleur dans le prompt
- Exercice : générer 3 clips de 5 secondes avec des mouvements de camera différents

3.5 **Prompt engineering pour la musique**
- Structure : Genre + Instruments + Tempo + Voix + Mood + Tags de section
- Tags de section : `[Verse]`, `[Chorus]`, `[Bridge]`, `[Outro]`
- Termes abstraits efficaces : "haunting", "uplifting", "ethereal", "gritty"
- Limite Suno : 1000 caracteres max pour le style prompt
- Exercice : générer 3 versions d'une même chanson dans 3 genres différents

3.6 **Le prompt chaining — Enchainer les modèles**
- Texte → Image → Video → Musique : un pipeline créatif complet
- Exemple : Claude ecrit un script → Midjourney généré les storyboards → Kling généré les clips → Suno généré la bande son
- Comment passer le contexte d'un modèle a l'autre
- Exercice : créer une sequence complète text-to-video avec musique

- Humor Level: [0-10]
- Formality: [0-10]
- Empathy: [0-10]
- Vocabulary: [technique, accessible, argot...]

## Core Mission
[En une phrase : ce pour quoi l'agent est optimise]

## Workflow
[Step-by-step : comment l'agent aborde chaque tache]

## Boundaries
- Will do: [liste de ce que l'agent accepte]
- Won't do: [liste de ce que l'agent refuse]
- Edge cases: [comment gerer les situations ambigues]

## Examples
[2-3 exemples d'interactions typiques avec l'agent]
