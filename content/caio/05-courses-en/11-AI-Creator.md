### MODULE 3 — Fundamental Prompt Engineering — The Art of Communicating with AI

**Objective**: The student masters the basics of prompting for ALL types of models.

**Lessons to generate:**

3.1 **Universal Prompting Principles**
- Specificity: the more précise you are, the better the result
- Structure: organize your instructions (context, task, format, constraints)
- Iteration: the first try is a draft, not the final result
- Examples: show what you want (few-shot learning)
- Negation: say what you do NOT want

3.2 **Prompt Engineering for Text**
- RACE Framework: Role + Action + Context + Expectation
- Chain-of-thought: "Think step by step"
- Few-shot: provide 2-3 examples of the desired format
- Reusable templates by content type
- Exercise: write 5 prompts for 5 content types (blog, email, script, social, newsletter)

3.3 **Prompt Engineering for Images**
- Structure: Subject + Environment + Style + Lighting + Camera
- Negative prompts: `--no hands, text, watermark, blurry`
- Prompt weights: `car::2 jungle::1` (Midjourney)
- Style références: `--sref` (Midjourney), style descriptions (DALL-E)
- Seed control: reproduce and vary a result
- Exercise: generate the same scene in 5 différent styles

3.4 **Prompt Engineering for Video**
- Describe FORCES, not appearances ("the wind blows the leaves" > "leaves that move")
- Camera keywords: dolly, tracking, crane, pan, tilt, orbit, zoom
- Specify 24fps for a cinematic look
- Lighting and color in the prompt
- Exercise: generate 3 clips of 5 seconds with différent camera movements

3.5 **Prompt Engineering for Music**
- Structure: Genre + Instruments + Tempo + Voice + Mood + Section Tags
- Section tags: `[Verse]`, `[Chorus]`, `[Bridge]`, `[Outro]`
- Effective abstract terms: "haunting", "uplifting", "ethereal", "gritty"
- Suno limit: 1000 characters max for the style prompt
- Exercise: generate 3 versions of the same song in 3 différent genres

3.6 **Prompt Chaining — Linking Models Together**
- Text → Image → Video → Music: a complète créative pipeline
- Example: Claude writes a script → Midjourney generates the storyboards → Kling generates the clips → Suno generates the soundtrack
- How to pass context from one model to another
- Exercise: create a complète text-to-video sequence with music

14.2 **The SOUL.md Framework — The Agent's Soul**
