# 🎬 Du Live au Produit : Guide Complet de Production Vidéo Faceless

## Le Shift Mental : Live vs Produit

---

### Ce qui change fondamentalement

```
LIVE (ce que tu sais faire)          PRODUIT (ce que tu apprends)
─────────────────────────────        ─────────────────────────────
Tu parles en temps réel          →   Tu scriptes AVANT de parler
Les erreurs restent              →   Les erreurs se coupent
1 prise = la vidéo finale       →   Multiples prises assemblées
L'énergie vient de l'instant    →   L'énergie vient du montage
Le rythme est naturel            →   Le rythme est CONSTRUIT
La durée est ce qu'elle est     →   Chaque seconde est intentionnelle
```

**La bonne nouvelle :** Tu sais déjà parler devant un micro. 
C'est le skill le plus dur à acquérir. Tout le reste c'est de la technique.

---

## LES 3 NIVEAUX DE PRODUCTION

Tu ne passes pas de "live brut" à "Fireship" en une semaine.
Voici la progression réaliste :

### 🟢 NIVEAU 1 : "Le Screen Recording Commenté" (Semaine 1-4)

C'est le format David Ondrej / Nate Herk / Nick Saraev.
Le plus proche de ce que tu fais déjà.

**Le concept :** Tu enregistres ton écran pendant que tu fais un truc, 
tu commentes par-dessus, tu coupes les moments morts.

**La différence avec le live :** Tu peux COUPER. C'est tout.
Ça change TOUT.

```
WORKFLOW NIVEAU 1 :

1. SCRIPT LÉGER (30 min)
   ├── Pas un script mot-à-mot
   ├── Un plan en bullet points :
   │   • Hook (ce qu'on va construire)
   │   • Étape 1, 2, 3...
   │   • Résultat final
   │   • CTA
   └── Tu sais OÙ tu vas, mais tu improvises le chemin

2. RECORDING (durée réelle du tuto + 20%)
   ├── Lance OBS
   ├── Enregistre ton écran
   ├── Parle normalement (comme en live)
   ├── Si tu te plantes → PAUSE → reprends
   │   (tu couperas en post)
   └── Montre le résultat final

3. POST-PROD SIMPLE (1-2h)
   ├── Importe dans DaVinci Resolve / CapCut
   ├── Coupe les silences et hésitations
   ├── Coupe les moments de chargement
   ├── Ajoute des zooms sur les parties importantes
   └── Exporte
```

**Outils Niveau 1 :**
| Outil | Prix | Rôle |
|-------|------|------|
| OBS Studio | Gratuit | Screen recording |
| DaVinci Resolve | Gratuit | Montage |
| Canva | Gratuit/Pro | Thumbnails |

**Exemple concret :** Tu montres comment builder un workflow n8n.
Tu enregistres ton écran, tu parles, tu coupes les temps morts.
C'est déjà 10x mieux qu'un live brut.

### 🟡 NIVEAU 2 : "Le Voix-Off Structuré" (Mois 2-3)

C'est le format Christophe Pauly / ColdFusion / LeMMMino.
La voix est enregistrée SÉPARÉMENT des visuels.

**Le game-changer :** Tu sépares l'AUDIO du VISUEL.
Ça te donne un contrôle total sur les deux.

```
WORKFLOW NIVEAU 2 :

1. SCRIPT COMPLET (1-2h)
   ├── Script mot-à-mot (avec Claude)
   ├── Marquer les moments visuels :
   │   [SCREEN: montrer n8n workflow]
   │   [GRAPHIC: schéma du système]
   │   [B-ROLL: timelapse ville]
   ├── Timer chaque section
   └── Relire à voix haute pour tester le flow

2. VOIX OFF (30-60 min)
   
   OPTION A : Tu enregistres toi-même
   ├── Micro correct (même un bon casque suffit)
   ├── Pièce calme
   ├── Lis le script section par section
   ├── Si tu te plantes → re-lis juste cette phrase
   └── Pas besoin d'être parfait (tu couperas)
   
   OPTION B : 11Labs avec ta voix clonée
   ├── Colle le script dans 11Labs
   ├── Ajuste les paramètres (speed, stability)
   ├── Génère section par section
   ├── Ré-génère les phrases qui sonnent bizarre
   └── Exporte chaque section en .mp3

3. VISUELS (1-2h)
   ├── Screen recording des démos (sans parler)
   ├── Screenshots annotés
   ├── B-roll depuis Pexels / Artgrid
   ├── Schémas simples (Canva / Excalidraw)
   └── Texte à l'écran pour les points clés

4. ASSEMBLAGE (2-3h)
   ├── Importe la voix off dans la timeline
   ├── Place les visuels PAR-DESSUS la voix
   ├── Chaque phrase = un visuel correspondant
   ├── Ajoute les transitions
   ├── Ajoute la musique de fond
   └── Exporte
```

**Outils Niveau 2 :**
| Outil | Prix | Rôle |
|-------|------|------|
| OBS Studio | Gratuit | Screen recording |
| DaVinci Resolve | Gratuit | Montage + color |
| 11Labs | ~22€/mois | Voix off clonée |
| Canva Pro | 12€/mois | Graphics + thumbnails |
| Epidemic Sound | 13€/mois | Musique de fond |
| Pexels | Gratuit | B-roll stock footage |

**Le truc qui change tout à ce niveau :**
Tu peux enregistrer ta voix off en pyjama à 23h.
Tu peux faire le screen recording le lendemain.
Tu peux assembler le surlendemain.
**Le live t'oblige à tout faire en même temps.
Le produit te libère.**

### 🔴 NIVEAU 3 : "Le Mini-Documentaire" (Mois 4+)

C'est le format Fireship / Kurzgesagt.
Chaque seconde est designée.

```
WORKFLOW NIVEAU 3 :

1. SCRIPT CINÉMATIQUE (2-3h)
   ├── Script avec Claude (skill youtube-script-creator)
   ├── Hook émotionnel en 5 secondes
   ├── Arc narratif en 3 actes
   ├── Loops de curiosité plantés
   ├── Chaque transition scriptée
   └── Storyboard simple (quel visuel pour quelle phrase)

2. VOIX OFF PREMIUM (1h)
   ├── 11Labs avec ta voix clonée
   ├── Paramètres affinés pour chaque émotion :
   │   • Hook : plus rapide, plus intense
   │   • Explication : calme, clair
   │   • Révélation : pause... puis énergie
   └── Post-traitement audio :
       EQ + compression + légère reverb

3. MOTION DESIGN (2-4h)
   ├── Text animations (CapCut Pro ou After Effects)
   ├── Schémas animés (Canva Animate ou Keynote)
   ├── Zoom dynamiques sur le screen recording
   ├── Transitions custom (pas les transitions par défaut)
   ├── Lower thirds et légendes stylées
   └── Effets de highlight/spotlight sur le code

4. SOUND DESIGN (30 min)
   ├── Musique de fond (Epidemic Sound)
   ├── Sound effects aux transitions
   │   (whoosh, click, pop)
   ├── Volume ducking automatique
   │   (musique baisse quand tu parles)
   └── Silence intentionnel aux moments clés

5. ASSEMBLAGE FINAL (2-3h)
   ├── Timeline multi-pistes :
   │   Piste 1 : Voix off
   │   Piste 2 : Musique
   │   Piste 3 : SFX
   │   Piste 4 : Screen recording
   │   Piste 5 : B-roll
   │   Piste 6 : Graphics/text
   ├── Color grading cohérent
   ├── Review complète
   └── Export final
```

**Outils Niveau 3 :**
| Outil | Prix | Rôle |
|-------|------|------|
| DaVinci Resolve | Gratuit | Montage principal |
| CapCut Pro | 8€/mois | Motion graphics faciles |
| After Effects | 24€/mois | Animations avancées (optionnel) |
| 11Labs Pro | 22€/mois | Voix off |
| Epidemic Sound | 13€/mois | Musique + SFX |
| Canva Pro | 12€/mois | Graphics |
| Pexels / Artgrid | Gratuit / 25€/mois | B-roll |
| Excalidraw | Gratuit | Schémas de systèmes |

## 📐 LE PROCESS COMPLET : DE L'IDÉE À LA PUBLICATION

Voici exactement ce que tu fais pour UNE vidéo, étape par étape :

### JOUR 1 : PRÉ-PRODUCTION (2h)

```
09:00 - 09:30 │ IDÉATION
               │ • Ouvre Claude, dis : "Voici le sujet : [X]. 
               │   Propose-moi 3 angles et 3 hooks."
               │ • Choisis l'angle qui te fait vibrer
               │ • Définis : Quel est le ONE THING 
               │   que le viewer retient ?
               │
09:30 - 10:30 │ SCRIPT
               │ • Utilise Claude avec le skill youtube-script-creator
               │ • Relis à voix haute
               │ • Chronomètre : 150 mots ≈ 1 minute parlée
               │   (donc 2000 mots ≈ 13 min de vidéo)
               │ • Marque dans le script :
               │   [SCREEN] = screen recording
               │   [GRAPH] = graphic/schéma
               │   [BROLL] = footage stock
               │   [TEXT] = texte à l'écran
               │
10:30 - 11:00 │ STORYBOARD RAPIDE
               │ • Pour chaque paragraphe du script,
               │   note quel visuel tu veux
               │ • Pas besoin de dessiner
               │ • Juste une liste :
               │   "0:00-0:15 : Animation titre + hook"
               │   "0:15-1:00 : Screen recording n8n"
               │   "1:00-1:30 : Schéma du système"
               │   etc.
```

### JOUR 2 : PRODUCTION (3-4h)

```
BLOC 1 : VOIX OFF (30-60 min)
─────────────────────────────
OPTION A - Tu enregistres toi-même :
│ • Ouvre Audacity (gratuit) ou ton DAW
│ • Micro branché, pièce calme
│ • Lis le script section par section
│ • Technique du "punch-in" :
│   Si tu te plantes sur une phrase,
│   fais 2 sec de silence, puis re-dis la phrase.
│   En post-prod tu coupes le raté.
│ • Exporte en WAV ou MP3 haute qualité
│
OPTION B - 11Labs :
│ • Colle le script dans 11Labs
│ • Voix = ton clone
│ • Génère par sections (pas tout d'un coup)
│ • Écoute chaque section
│ • Ré-génère ce qui sonne pas naturel
│ • Astuce : ajoute "..." pour des pauses
│   et des virgules pour le rythme
│ • Exporte chaque section

BLOC 2 : SCREEN RECORDING (1-2h)
─────────────────────────────────
│ • Ouvre OBS Studio
│ • Résolution : 1920×1080 minimum (2560×1440 idéal)
│ • FPS : 30 (suffisant pour des tutos)
│ • IMPORTANT : NE PARLE PAS pendant le recording
│   La voix off est déjà faite séparément
│ • Fais tes démos calmement
│ • Si tu te plantes : pas grave, refais juste ce passage
│ • Enregistre PLUS que nécessaire (tu couperas)
│ • Astuce OBS : utilise les "Scenes" pour switcher
│   entre ton écran entier et des zones zoomées

BLOC 3 : ASSETS VISUELS (30-60 min)
────────────────────────────────────
│ • Canva : crée les schémas/diagrammes
│ • Excalidraw : dessine les architectures de systèmes
│ • Pexels : télécharge 5-10 clips de b-roll
│   (villes, tech, abstract, timelapse)
│ • Screenshots : capture les résultats finaux
│ • Exporte tout dans un dossier "assets"
```

### JOUR 3 : POST-PRODUCTION (2-4h)

```
ÉTAPE 1 : SETUP DU PROJET (15 min)
───────────────────────────────────
│ • Ouvre DaVinci Resolve (ou CapCut)
│ • Crée un nouveau projet
│ • Importe tous les fichiers :
│   📁 Voix off (.mp3/.wav)
│   📁 Screen recordings (.mp4)
│   📁 B-roll clips (.mp4)
│   📁 Graphics (.png)
│   📁 Musique (.mp3)

ÉTAPE 2 : LA VOIX OFF = TA COLONNE VERTÉBRALE (15 min)
──────────────────────────────────────────────────────
│ • Place la voix off sur la Piste 1
│ • C'est elle qui dicte TOUT le rythme
│ • Coupe les silences trop longs
│ • Coupe les erreurs / reprises
│ • Tu obtiens une voix off propre et fluide
│ 
│ C'est LA différence fondamentale avec le live :
│ en live, le visuel dicte la voix.
│ En produit, LA VOIX DICTE LE VISUEL.

ÉTAPE 3 : PLACE LES VISUELS (1-2h)
───────────────────────────────────
│ • Écoute ta voix off phrase par phrase
│ • Pour CHAQUE phrase, place le visuel correspondant :
│   
│   "Ce workflow n8n..."       → [Screen recording du workflow]
│   "...connecte 3 APIs..."   → [Zoom sur les nodes]
│   "...et le résultat..."    → [Screenshot du résultat]
│   "Imagine un système..."   → [B-roll tech / schéma animé]
│
│ RÈGLE D'OR : Change de visuel toutes les 5-8 secondes
│ C'est ce qui donne le rythme "Fireship"
│ Les viewers décrochent sur des plans statiques > 10 sec

ÉTAPE 4 : DYNAMISE (30-60 min)
─────────────────────────────
│ • ZOOMS : Sur les parties importantes du screen recording
│   (DaVinci : Alt+clic pour zoom, ou keyframes scale)
│ • TEXT ON SCREEN : Les mots-clés importants apparaissent
│ • TRANSITIONS : Coupes franches (pas de fondus fancy)
│ • HIGHLIGHTS : Cercle ou rectangle sur les zones 
│   de l'écran où regarder
│ • SPEED : Accélère les moments de chargement/installation

ÉTAPE 5 : AUDIO FINAL (15-30 min)
──────────────────────────────────
│ • Musique de fond sur Piste 2
│   Volume : -20dB sous la voix (elle doit être subtile)
│ • Sound effects aux transitions (optionnel mais pro)
│ • Vérifie que la voix est TOUJOURS claire et dominante
│ • Si 11Labs : un peu d'EQ pour réchauffer
│   (boost léger 200Hz, cut 5kHz harsh)

ÉTAPE 6 : REVIEW + EXPORT (15 min)
───────────────────────────────────
│ • Regarde la vidéo EN ENTIER
│ • Vérifie : le hook donne envie de rester ?
│ • Vérifie : pas de silence mort > 2 secondes ?
│ • Vérifie : les visuels changent assez souvent ?
│ • Export : MP4, H.264, 1080p, audio 320kbps
```

## 🔧 SETUP OBS POUR LE SCREEN RECORDING

```
PARAMÈTRES OBS RECOMMANDÉS
──────────────────────────

Settings > Output :
├── Mode : Simple
├── Recording Quality : Indistinguishable Quality
├── Recording Format : MKV (puis remux en MP4 après)
└── Encoder : Hardware si dispo (NVENC/AMD)

Settings > Video :
├── Base Resolution : 2560×1440 (ou 1920×1080)
├── Output Resolution : 1920×1080
└── FPS : 30

Sources à configurer :
├── "Display Capture" → Ton écran entier
├── "Window Capture" → App spécifique (n8n, VS Code...)
└── "Audio Output Capture" → Son du système (optionnel)

NE PAS capturer ton micro dans OBS
(la voix off se fait séparément)

Hotkeys à configurer :
├── Start Recording : F9
├── Stop Recording : F10
└── Pause : F11
```

## 🎙️ SETUP 11LABS POUR LA VOIX OFF

```
CONFIGURATION RECOMMANDÉE
─────────────────────────

1. CLONE TA VOIX
   ├── Enregistre 30-60 min de ta voix naturelle
   │   (parle de sujets variés, avec émotion)
   ├── Uploade dans 11Labs > Voice Lab > Add Voice
   ├── "Professional Voice Clone" (meilleure qualité)
   └── Teste avec quelques phrases

2. PARAMÈTRES DE GÉNÉRATION
   ├── Model : Eleven Multilingual v2
   ├── Stability : 0.55 (naturel mais pas trop aléatoire)
   ├── Similarity : 0.80 (fidèle à ta voix)
   ├── Style : 0.25 (léger dynamisme)
   └── Speed : 1.05-1.10 (légèrement plus rapide = énergie)

3. TECHNIQUE D'ÉCRITURE POUR 11LABS
   
   ❌ "L'intelligence artificielle transforme notre monde"
   ✅ "L'intelligence artificielle... transforme notre monde."
   
   ❌ "Voici les 3 étapes : premièrement, deuxièmement..."
   ✅ "Voici les 3 étapes.
       Premièrement...
       Deuxièmement..."
   
   Les retours à la ligne = pauses naturelles
   Les "..." = micro-pauses
   Les phrases courtes = meilleur rendu

4. WORKFLOW DE GÉNÉRATION
   ├── Génère le script SECTION PAR SECTION
   │   (pas tout d'un coup)
   ├── Écoute chaque section
   ├── Ré-génère si ça sonne robotique
   ├── Exporte chaque section en MP3
   └── Assemble dans DaVinci Resolve
```

## 🎨 L'ART DU MONTAGE : CE QUI FAIT LA DIFFÉRENCE

### Les 7 Techniques qui séparent l'amateur du pro

```
1. LA COUPE EN J ET EN L (J-CUT / L-CUT)
   ──────────────────────────────────────
   J-Cut : L'audio de la prochaine scène commence 
   AVANT que le visuel change.
   → Crée une transition fluide et naturelle.
   
   L-Cut : L'audio de la scène actuelle CONTINUE
   pendant que le visuel a déjà changé.
   → Garde l'attention pendant les transitions.
   
   C'est ce que font Christophe Pauly et tous les
   documentaristes. Ça rend le montage invisible.

2. LE ZOOM DYNAMIQUE
   ──────────────────
   Quand tu montres du code ou un workflow :
   • Vue d'ensemble → Zoom progressif sur la zone importante
   • Durée du zoom : 0.5-1 seconde
   • DaVinci : keyframes sur "Zoom" dans l'Inspector
   • C'est la signature de Fireship

3. LE RYTHME 5-8 SECONDES
   ────────────────────────
   Change de visuel toutes les 5-8 secondes.
   Pas besoin que le visuel soit spectaculaire.
   Même un léger zoom ou un angle différent suffit.
   
   C'est pour ça que Fireship feels fast :
   pas parce qu'il parle vite,
   mais parce que les visuels changent souvent.

4. LE TEXT ON SCREEN
   ─────────────────
   Quand tu dis un mot-clé important :
   → Affiche-le en texte à l'écran
   
   "Ce SYSTÈME utilise trois APIs..."
   → Le mot "3 APIs" apparaît en gros
   
   Police recommandée : Inter, Montserrat, ou SF Pro
   Couleur : Blanc sur fond sombre, ou ta couleur accent

5. LE HIGHLIGHT / SPOTLIGHT
   ────────────────────────
   Sur un screen recording complexe :
   → Assombris tout SAUF la zone d'intérêt
   → Ou ajoute un rectangle/cercle coloré
   → Guide l'œil du viewer exactement là où tu veux

6. LE SPEED RAMP
   ──────────────
   Les moments ennuyeux (installations, chargements) :
   → Accélère ×2 ou ×4
   → Ajoute un son "whoosh" ou une musique upbeat
   → Le viewer comprend qu'on skip, mais il reste

7. LE SILENCE INTENTIONNEL
   ────────────────────────
   AVANT une révélation importante :
   → 0.5-1 seconde de silence total
   → Puis la phrase clé
   
   C'est contre-intuitif (en live tu remplis les silences)
   mais en produit, le silence CRÉE de la tension.
```

## 📋 CHECKLIST PAR VIDÉO

```
PRÉ-PRODUCTION
□ Sujet choisi + angle défini
□ Script écrit (ou bullet points niveau 1)
□ Storyboard rapide (quel visuel quand)
□ Assets listés (screenshots, b-roll, schémas à créer)

PRODUCTION  
□ Voix off enregistrée (toi ou 11Labs)
□ Screen recordings faits (sans micro)
□ B-roll téléchargé
□ Graphics/schémas créés

POST-PRODUCTION
□ Voix off nettoyée et placée
□ Visuels synchronisés avec la voix
□ Zooms + highlights ajoutés
□ Text on screen pour les mots-clés
□ Musique de fond ajoutée (-20dB)
□ Transitions propres
□ Review complète (regarder en entier)
□ Export final (1080p, H.264)

PUBLICATION
□ Thumbnail créée (Canva)
□ Titre optimisé (avec mot-clé principal)
□ Description avec timestamps
□ Tags ajoutés
□ Première phrase de description = hook
□ Lien communauté en description
□ Programmée au bon jour/heure

REPURPOSE
□ Audio extrait → Podcast
□ Script → Article newsletter
□ Meilleurs moments → Shorts
□ Insights clés → Tweets
```
