## Schedule Dedie

Le temps Agentik OS est sanctuarise sur le weekend. Samedi et dimanche, de 11h a 18h, soit sept heures par jour. Cela represente quatorze heures par semaine, cinquante-six heures par mois, et six cent soixante-douze heures par an.

Le weekend est sacre : deep work sur la CAIO Academy (curriculum, contenu, communaute) et les apps en cours. Pas de calls clients, pas de reunions. Pure creation.

## Stack Technique Standard

**Frontend.** Next.js 15 avec App Router, TypeScript, Tailwind CSS avec Shadcn/UI, et Framer Motion pour les animations.

**Backend.** Convex pour la database et les functions, Clerk pour l'authentification, Stripe pour les paiements, et Vercel pour l'hosting.

**AI.** Claude Code (Anthropic), Composio pour les automations, custom agents.

**Mobile (si necessaire).** Expo et React Native avec EAS Build.

Cette stack est aussi celle enseignee dans la CAIO Academy. Les etudiants apprennent a deployer avec les memes outils que Gareth utilise en production.
