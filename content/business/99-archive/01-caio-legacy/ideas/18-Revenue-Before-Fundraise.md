## Le Concept "Default Alive" de Paul Graham

### Definition

```
DEFAULT ALIVE = tes revenus couvriront tes couts AVANT
que tu manques d'argent, meme sans lever.

DEFAULT DEAD = si tu ne leves pas, tu meurs.

FORMULE :
Si Revenue_mensuel > Depenses_mensuelles : DEFAULT ALIVE
Si Revenue_mensuel < Depenses_mensuelles :
    Si Revenue croit assez vite pour croiser les depenses
    AVANT que le cash soit a zero : DEFAULT ALIVE
    Sinon : DEFAULT DEAD
```

### Pourquoi c'est Important

| Status | Consequence sur la Levee |
|--------|--------------------------|
| Default Alive | Tu leves PAR CHOIX. Les VCs le sentent. Tu negocies en force. |
| Default Dead | Tu leves PAR NECESSITE. Les VCs le sentent aussi. Ils dictent les termes. |

```
LA PSYCHOLOGIE DU VC :

"Ce fondateur a 30K MRR et croit de 15%/mois.
 Il n'a PAS BESOIN de nous. S'il leve, c'est pour ACCELERER.
 On doit faire une offre attractive sinon il part chez un autre fond
 OU il continue sans nous."

vs

"Ce fondateur a 0 revenue et brule 15K/mois.
 Il a besoin de nous. On peut prendre notre temps.
 On peut negocier des termes plus favorables."

QUELLE POSITION VEUX-TU ETRE?
```

## Quand Lever EST la Bonne Decision

| Signal | Action |
|--------|--------|
| Ton marche a des effets reseau forts | Lever pour acquerir vite (winner-take-all) |
| Un concurrent bien finance attaque | Lever pour ne pas se faire ecraser |
| Tu as besoin d'infra couteuse (GPU, data) | Lever car les couts depassent les revenus |
| Le timing est critique (fenetre de marche) | Lever pour accelerer avant que la fenetre se ferme |
| Tu veux construire une equipe senior | Lever car les salaires depassent le MRR |

### Quand NE PAS Lever

| Signal | Action |
|--------|--------|
| Pas de PMF encore | Bootstrap plus longtemps |
| Marche de niche (< 500M TAM) | Reste bootstrap, optimise la marge |
| Tu valorises la liberte au-dessus de la croissance | Reste bootstrap |
| Tu n'as pas de plan clair pour l'argent | Ne leve pas "au cas ou" |
| Tu es en default alive confortable | Ne dilue pas pour le fun |
