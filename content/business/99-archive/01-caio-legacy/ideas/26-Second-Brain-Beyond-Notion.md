## Vision

Le Personal Knowledge Management (PKM) est casse. Notion, Obsidian, Roam, Logseq - tous partent du meme principe : tu ecris, tu organises, tu retrouves. Meme avec l'IA (Notion AI, Mem), ca reste des outils de STOCKAGE avec une couche de generation.

Le vrai second brain ne stocke pas. Il PENSE. Il fait ce que ton cerveau devrait faire mais n'a pas la bande passante pour :

1. Detecter les themes recurrents dans tes notes
2. Connecter des idees que tu as ecrites a 3 mois d'ecart
3. Synthetiser ta pensee sur un sujet
4. Te proposer des actions basees sur tes propres reflexions
5. Te montrer l'evolution de ta pensee dans le temps

## Fonctionnalites Principales

### 1. Capture Universelle (Zero Friction)

| Methode | Comment |
|---------|---------|
| Voix | Enregistrement -> transcription -> structuration auto |
| Texte rapide | Input minimaliste, pas de formatage requis |
| Web clip | Extension navigateur, capture en 1 clic |
| Screenshot | OCR automatique + contexte |
| Import | Notion, Obsidian, Roam, Google Docs |
| Email forward | Transferer un email -> capture auto |

Principe : la capture doit couter ZERO effort cognitif. Pas de dossier a choisir, pas de tag a mettre. L'IA organise.

### 2. Detection de Patterns

L'IA analyse en continu l'ensemble de tes notes et detecte :

```
+-- PATTERN DETECTE -----------------------------------------------+
|                                                                   |
|  THEME RECURRENT : "Autonomie dans le travail"                   |
|                                                                   |
|  Tu as ecrit sur ce sujet 7 fois en 3 mois :                    |
|  - 12 Jan : reflexion sur pourquoi tu detestes le micromanagement|
|  - 28 Jan : note sur les avantages du freelance                 |
|  - 3 Feb : citation de Naval sur le leverage                    |
|  - 15 Feb : frustration avec un client controlant               |
|  - 22 Feb : idee de produit SaaS (= pas de client)              |
|  - 8 Mar : article sauvegarde sur le solopreneurship            |
|  - 14 Mar : reflexion sur ta definition du succes               |
|                                                                   |
|  SYNTHESE : L'autonomie semble etre ta valeur #1 dans le        |
|  travail. Ton irritation augmente quand elle est menacee.        |
|  Ton cerveau te pousse clairement vers un modele solo/produit.  |
|                                                                   |
|  ACTION SUGGEREE : Definir ce que "autonomie totale" signifie   |
|  concretement pour toi, et l'utiliser comme filtre decisif      |
|  pour tes prochains choix de projets.                            |
+------------------------------------------------------------------+
```

### 3. Decouverte de Connexions

```
+-- CONNEXION DECOUVERTE ------------------------------------------+
|                                                                   |
|  NOTE A (3 Janvier) : "Le stoicisme dit de se concentrer        |
|  uniquement sur ce qu'on controle"                               |
|                                                                   |
|      <---- CONNEXION ---->                                        |
|                                                                   |
|  NOTE B (28 Fevrier) : "Mon anxiete vient du fait que           |
|  j'essaie de controler les reactions des clients"                |
|                                                                   |
|  INSIGHT : Ta lecture stoicienne en janvier anticipe exactement  |
|  le probleme que tu identifies en fevrier. Appliquer le          |
|  "dichotomie du controle" a tes relations clients pourrait      |
|  reduire ton anxiete de facon significative.                     |
+------------------------------------------------------------------+
```

### 4. Propositions d'Actions

Basees UNIQUEMENT sur tes propres ecrits :

| Type | Exemple |
|------|---------|
| Synthese a ecrire | "Tu as 12 notes sur le minimalisme. Veux-tu que je cree une synthese ?" |
| Decision a prendre | "Tu mentionnes le freelance depuis 3 mois. Il est peut-etre temps de faire un plan concret." |
| Contradiction a resoudre | "Tu dis vouloir plus de temps libre MAIS tu prends des projets supplementaires." |
| Recherche a approfondir | "Tu as pose 3 questions sans reponse sur l'investissement immobilier." |
| Habitude a former | "Tu notes regulierement que la meditation t'aide. Tu ne medites pourtant que 2x/semaine." |

### 7. Recherche Semantique

Pas de mots-cles. Des questions en langage naturel :
- "Qu'est-ce que je pense du management ?"
- "Quels sont mes blocages recurrents ?"
- "Qu'est-ce que j'ai appris sur l'investissement ?"
- "Quand est-ce que je suis le plus heureux ?"

### 8. Mode Conversation avec Ton Second Brain

Discuter avec l'IA MAIS dans le contexte de TOUTES tes notes :
- "Qu'est-ce que mes notes disent sur ma relation avec l'argent ?"
- "Y a-t-il des contradictions dans ce que j'ecris ?"
- "Base sur tout ce que tu sais de moi, quel devrait etre mon prochain projet ?"
