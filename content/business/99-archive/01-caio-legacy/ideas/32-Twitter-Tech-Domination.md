| Element | Specification | Exemple |

| Banner | Proposition de valeur visuelle + URL/CTA | "Je construis des agents IA -- Rejoins 5K builders" |

```
LIGNE 1: Ce que tu fais + pour qui (proposition de valeur)
LIGNE 2: Preuve sociale ou resultat concret
LIGNE 3: CTA (newsletter, communaute, produit)

Exemple:
"Agents IA pour entrepreneurs solo | 50+ systemes deployes
Fondateur @AgentikOS | Ex-dev -> solopreneur IA
Newsletter: [lien] (5K builders)"
```

## Mix de Contenu

### Repartition Optimale

| Type | Part | Frequence | Objectif |
|------|------|-----------|----------|
| Tweets d'insight | 50% | 2-3/jour | Autorite + impressions |
| Threads | 30% | 2/semaine | Viralite + followers |
| Engagement (replies) | 10% | 10-20/jour | Croissance rapide |
| Personnel / behind the scenes | 10% | 3-4/semaine | Connexion humaine |

### Tweets d'Insight: Les Regles

```
STRUCTURE:
- 1 a 3 phrases maximum
- Opinion tranchee (pas de "peut-etre", "je pense que")
- Formule: Observation contrariante + Conclusion actionnable

EXEMPLES:
"Les agents IA ne remplacent pas les employes.
Ils remplacent les process mal concu.
La difference est fondamentale."

"Tout le monde parle de prompts.
Personne ne parle de systemes.
Les prompts sont des outils.
Les systemes sont des businesses."

"Le freelance IA a 150 EUR/h est facile.
Le difficile c'est de refuser les clients a 50 EUR/h."
```

### Threads: Formules Qui Marchent

| Formule | Structure | Engagement Moyen |
|---------|-----------|-----------------|
| "X choses que j'ai apprises en..." | Hook -> liste numerotee -> CTA | Eleve |
| "Comment j'ai construit X en Y" | Hook -> etapes -> resultat -> lecon | Tres eleve |
| "La verite sur Z" | Hook contrarian -> arguments -> conclusion | Eleve |
| "Thread gratuit: [sujet]" | Hook valeur -> contenu dense -> CTA | Moyen-eleve |
| "X outils/ressources pour Y" | Hook -> liste curatee -> bonus | Tres eleve (sauvegardes) |
| "Mon process exact pour Z" | Hook resultat -> etapes detaillees -> template | Eleve |
| "Avant/Apres: X" | Hook transformation -> details -> lecons | Moyen-eleve |

### Structure d'un Thread Performant

```
TWEET 1 (HOOK): Promesse forte + raison de lire
  - "J'ai passe 200h a tester des agents IA.
     Voici les 7 qui valent vraiment le coup.
     (Thread)"

TWEETS 2-8 (CORPS): 1 point = 1 tweet
  - Chaque tweet est autonome (valeur seule)
  - Format: Titre court + explication + detail concret
  - Images/screenshots quand possible

TWEET 9 (RECAPITULATIF): Resume + CTA
  - "Recap:
     1. [Point 1]
     2. [Point 2]
     ...
     Si tu veux aller plus loin: [lien newsletter]
     Like + RT si c'est utile."
```

## La Strategie "Reply Guy" (Croissance la Plus Rapide)

### Principe

Commenter avec du contenu de valeur sur les tweets de gros comptes = visibilite gratuite aupres de LEUR audience.

### Comment Faire

```
ETAPE 1: Identifier 10-20 comptes cibles
  - Meme niche (IA, tech, solopreneuriat)
  - 10K-500K followers (assez gros pour le reach, pas trop pour etre noye)
  - Actifs (postent 1-3x/jour)

ETAPE 2: Activer les notifications pour ces comptes

ETAPE 3: Repondre dans les 15 premieres minutes
  - Etre parmi les 3-5 premieres reponses
  - JAMAIS de "Great post!" ou "Love this"
  - TOUJOURS ajouter de la valeur

ETAPE 4: Format de reply qui convertit
  - Ajouter un point que le tweet original n'a pas mentionne
  - Partager une experience personnelle pertinente
  - Poser une question intelligente qui genere discussion
```

### Exemples de Replies de Valeur

| Tweet Original | Mauvaise Reply | Bonne Reply |
|---------------|----------------|-------------|
| "Les agents IA vont changer le business" | "Tellement vrai!" | "J'ai deploye 12 agents pour un client PME le mois dernier. Resultat: -40h/semaine de taches repetitives. Le point que personne ne mentionne: le vrai ROI est dans la qualite des decisions, pas le temps gagne." |

## Erreurs a Eviter

| Erreur | Consequence | Solution |
|--------|------------|----------|
| Poster sans engagement | Compte invisible | 70 replies/semaine minimum |
| Contenu generique | Zero viralite | Opinions tranchees, donnees concretes |
| Trop de promotion | Unfollow massif | Ratio 90% valeur / 10% promo |
| Ignorer les DMs | Perte de leads | Repondre sous 24h aux DMs pertinents |
| Poster en francais uniquement | Marche limite | 60% FR / 40% EN minimum |
| Etre "nice" tout le temps | Aucun engagement | Opinions polarisantes (mais argumentees) |
