## La Progression : De Zero a 100K Abonnes

### Phase 1 : Fondations (Mois 1-6)

```
Objectif : 1000 abonnes YouTube + 5000 followers Twitter

Actions :
|-- 2 videos YouTube / semaine (court, 8-12 min)
|-- 1 thread Twitter / jour
|-- 1 article LinkedIn / semaine
|-- Commenter sur 20 posts/jour (Twitter)
|-- Repondre a TOUTES les questions (commentaires)
+-- Budget : 0 EUR (juste du temps)

Revenue attendu : 0-500 EUR/mois (AdSense insignifiant)
Temps investi : 15-20h/semaine
```

### Phase 2 : Traction (Mois 6-12)

```
Objectif : 10K abonnes YouTube + 25K followers Twitter

Actions :
|-- Videos plus longues et profondes (15-25 min)
|-- Premiers collabs avec d'autres createurs
|-- Premiers sponsorships (1-2K/video)
|-- Lancer un newsletter (Substack/Beehiiv)
|-- Premier mini-cours (47-97 EUR)
+-- Investir en equipement : camera, micro, eclairage (2-3K EUR)

Revenue attendu : 3-8K EUR/mois
Temps investi : 20-25h/semaine
```

### Phase 3 : Monetisation (Mois 12-18)

```
Objectif : 30K abonnes YouTube + 50K followers Twitter

Actions :
|-- Sponsorships reguliers (5-10K/video)
|-- Cours complet lance (297 EUR)
|-- Communaute payante lancee (27-67 EUR/mois)
|-- Premiers speaking gigs
|-- Deleguer le montage video
+-- Equipe : 1 monteur (1-2K/mois), 1 community manager (partiel)

Revenue attendu : 15-35K EUR/mois
Temps investi : 15-20h/semaine (delegation)
```

### Phase 4 : Domination (Mois 18-36)

```
Objectif : 100K+ abonnes YouTube + 100K+ followers Twitter

Actions :
|-- Programme premium (997-2000 EUR)
|-- Conferences internationales
|-- Proposition de livre (editeur)
|-- B2B : workshops entreprises
|-- Embaucher : equipe de 3-5 personnes
+-- Explorer : podcast propre, media company

Revenue attendu : 50-150K EUR/mois
Temps investi : 10-15h/semaine (leadership, pas execution)
```
