# Conseiller Financier IA pour les 99%

> La gestion de patrimoine, reservee aux riches, democratisee par l'IA.
> Pas un robo-advisor qui choisit des ETFs.
> Un vrai conseiller strategique : epargne, investissement, impots, dette, scenarios.

---

## Vision

Le conseil financier de qualite est reserve aux gens qui ont deja de l'argent. Un conseiller en gestion de patrimoine demande 100K+ d'actifs minimum et facture 1-2% par an. Pour les 99%, il reste YouTube, Reddit, et l'intuition.

Conseiller Financier IA democratise le conseil strategique. Pas le simple "mets ton argent dans un ETF World" (ca, les robo-advisors le font deja). Mais le conseil complet : combien epargner, ou investir, comment optimiser les impots, quel est le plan pour la retraite, et surtout : "si je fais X, que se passe-t-il dans 5 ans ?"

## Fonctionnalites Principales

### 1. Analyse Revenus / Depenses

Import automatique via Open Banking ou saisie manuelle :

```
+-- ANALYSE FINANCIERE MENSUELLE ---------------------------------+
|                                                                  |
|  REVENUS : 4 200 EUR net                                        |
|  DEPENSES : 3 100 EUR                                           |
|  EPARGNE : 1 100 EUR (26% - objectif 30%)                      |
|  [========================------] 87% de l'objectif              |
|                                                                  |
|  REPARTITION DEPENSES :                                          |
|  Logement        [==============------]  1 200 EUR  (39%)       |
|  Alimentation    [========------------]    450 EUR  (15%)       |
|  Transport       [=====---------------]    280 EUR  (9%)        |
|  Loisirs         [=======--------------]    350 EUR  (11%)       |
|  Abonnements     [====----------------]    180 EUR  (6%)        |
|  Divers          [=============-------]    640 EUR  (21%)       |

|  [!] "Divers" a 21% est trop eleve. Detailler pour optimiser.  |
|  [+] Taux d'epargne de 26% : au-dessus de la moyenne FR (15%). |
+------------------------------------------------------------------+
```

### 3. Strategie d'Investissement

Pas de gestion automatique (ca c'est un robo-advisor). Du CONSEIL strategique :

| Profil | Allocation Recommandee |
|--------|----------------------|
| Prudent | 70% fonds euros + 20% ETF obligataire + 10% ETF actions |
| Equilibre | 40% fonds euros + 40% ETF actions + 20% SCPI |
| Dynamique | 20% fonds euros + 60% ETF actions + 20% crypto/PE |
| Agressif | 80% ETF actions + 15% crypto + 5% startup |

L'IA adapte selon :
- Age, horizon d'investissement
- Situation familiale
- Tolerance au risque (questionnaire)
- Objectifs (retraite, achat immo, FIRE)
- Enveloppes disponibles (PEA, AV, CTO)

### 4. Optimisation Fiscale

Specifique au pays de l'utilisateur (France en V1) :

| Levier | Economie Potentielle | Applicable |
|--------|---------------------|------------|
| Maximiser PEA (exo 5 ans) | 30% flat tax evitee | Si PEA non plein |
| Assurance-vie (abattement 8 ans) | 4 600 EUR d'abattement | Si > 8 ans |
| Don a association | 66% credit d'impot | Si dons reguliers |
| Investissement PME | 25% credit d'impot | Si appetence risque |
| Defiscalisation immobiliere | Variable (Pinel, Denormandie) | Si projet immo |

### 5. Plan de Remboursement de Dettes

```
+-- STRATEGIE DETTE -----------------------------------------------+
|                                                                   |
|  Credit auto : 8 000 EUR a 4.5%                                 |
|  Mensualite actuelle : 350 EUR                                   |
|  Fin prevue : Novembre 2027                                      |
|                                                                   |
|  OPTION A : Continuer normalement                                 |
|  -> Cout total des interets : 920 EUR                            |
|                                                                   |
|  OPTION B : Rembourser 200 EUR/mois en plus                      |
|  -> Fin en Mai 2026 (18 mois avant)                              |
|  -> Economie d'interets : 480 EUR                                |
|  -> Impact : -200 EUR/mois d'epargne pendant 14 mois            |
|                                                                   |
|  RECOMMANDATION : Option B si ton fonds d'urgence > 3 mois      |
|  (Tu as 4.2 mois -> Option B recommandee)                       |
+------------------------------------------------------------------+
```

### 6. Modelisation "What-If"

Le differenciateur cle. Questions possibles :

| Scenario | Reponse |
|----------|---------|
| "Si j'achete un appart a 250K ?" | Simulation mensualites, impact patrimoine, date rentabilite |
| "Si je passe freelance ?" | Estimation revenus, filet de securite necessaire, timeline |
| "Si j'investis 500/mois pendant 20 ans ?" | Courbe de croissance, scenarios bull/bear/base |
| "Quand est-ce que je peux prendre ma retraite ?" | Age FIRE selon taux d'epargne et rendements |
| "Si je perds mon emploi ?" | Autonomie financiere en mois, actions immediates |

### 7. Alertes et Rappels

| Alerte | Frequence |
|--------|-----------|
| Livret A plein (22 950 EUR) -> reorienter | Ponctuelle |
| Depense anormale detectee | Temps reel |
| Objectif d'epargne en danger | Mensuelle |
| Opportunite fiscale (fin d'annee) | Annuelle |
| Rebalancing portfolio recommande | Trimestrielle |

### 8. Education Financiere Contextuelle

Pas des cours generiques. Des explications AU MOMENT ou tu en as besoin :
- Tu ouvres un PEA ? Explication en 30 secondes de ce que c'est et pourquoi c'est optimal.
- Tu investis en SCPI ? Avantages, risques, fiscalite en 1 minute.
- Tu herites ? Droits de succession, options, timeline.

## Risques et Reglementation

| Risque | Impact | Mitigation |
|--------|--------|------------|
| Reglementation CIF (Conseil en Investissement) | Haut | Positionnement "education", pas de conseil reglemente |
| Responsabilite si mauvais conseil | Haut | Disclaimers clairs, pas de gestion directe |
| Open Banking limites d'API | Moyen | Saisie manuelle en fallback |
| RGPD donnees financieres | Haut | Chiffrement, pas de stockage cote serveur des credentials |
| Concurrence des neobanques | Moyen | Specialisation conseil, pas banking |
