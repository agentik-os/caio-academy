## L'Homme qui a Transforme un VPS a 72 EUR en Empire de 3.2 Millions

En mars 2026, Gareth Moison vivait sur un serveur VPS a 72.61 euros par mois. Il codait avec Claude, un assistant IA d'Anthropic, et possedait exactement zero euros de revenus recurents. Quatre ans plus tard, son ecosysteme genere 3.2 millions d'euros par an, emploie 11 personnes, et a decline trois offres d'acquisition.

Ce n'est pas une success story Silicon Valley. Pas de levee de fonds spectaculaire, pas de pitch deck viralise, pas de mentor milliardaire. C'est l'histoire d'un developpeur ADHD de 33 ans qui a compris, avant presque tout le monde, que l'ere de l'IA ne recompenserait pas ceux qui utilisent l'IA, mais ceux qui pensent avec.

---

## Acte I : L'Obscurite (Mars - Juin 2026)

### Le Point de Depart

Quand on lui demande de decrire sa situation en mars 2026, Moison ne romantise rien.

"J'avais un VPS, une collection de fichiers markdown qui organisaient ma vie, et une conviction que 86% des gens utilisaient l'IA comme des idiots. Pas meilleur terme. Ils copiaient-collaient dans ChatGPT et acceptaient la premiere reponse. Je savais que ce n'etait pas comme ca qu'on creait de la valeur."

```
+------------------------------------------------------------------+
|  MARS 2026 : ETAT DES LIEUX                                     |
|                                                                  |
|  Actifs :                                                        |
|  |-- Competences techniques (full-stack, cybersecurite)          |
|  |-- Systeme personnel organise (1-Life, 200+ fichiers .md)     |
|  |-- Connaissance profonde de Claude et des LLMs                 |
|  |-- Rien d'autre.                                               |
|                                                                  |
|  Passifs :                                                       |
|  |-- 0 audience                                                  |
|  |-- 0 revenu recurent                                           |
|  |-- 0 employe                                                   |
|  |-- Loyer VPS : 72.61 EUR/mois                                 |
|  +-- Charges : ~1 500 EUR/mois minimum                          |
+------------------------------------------------------------------+
```

Sa premiere video YouTube, publiee le 14 avril 2026, s'intitulait "Pourquoi tu utilises ChatGPT comme un amateur (et comment les 14% pensent vraiment)." Elle a fait 47 vues en une semaine.

"47 vues," repete-t-il en souriant. "Ma mere en comptait probablement pour 12."

Mais il y avait un detail : le taux de retention etait de 73%. Dans le monde YouTube, c'est exceptionnel. Les 47 personnes qui regardaient allaient jusqu'au bout. Le message resonait. Le probleme etait la distribution, pas le contenu.

### Le Premier Client (Mai 2026)

Le tournant initial n'est pas venu de YouTube mais de LinkedIn. Un post sur "Comment j'utilise l'IA pour remplacer une equipe de 5 consultants" avait ete repris par un DRH de PME lyonnaise.

"Il m'a contacte en DM. Il voulait que je forme ses 30 managers a utiliser l'IA. J'ai dit oui, j'ai invente le programme en 48 heures avec Claude, et j'ai facture 3 000 euros. C'est le moment ou j'ai realise que les gens payaient pour apprendre a PENSER avec l'IA, pas pour des prompts."

| Mois | Evenement | Revenue | Lecon |
|------|-----------|---------|-------|
| Avril 2026 | Premiere video YouTube (47 vues) | 0 | Le contenu est bon, la distribution non |
| Mai 2026 | Premier client consulting (LinkedIn) | 3 000 EUR | Les gens paient pour la metacognition |
| Juin 2026 | 3 clients, 1 cours en dev | 8 000 EUR | Le B2B est le meilleur canal |


"Ce thread a ete le moment ou tout a bascule. Pas parce qu'il etait viral, mais parce qu'il a attire les BONNES personnes. Des DRH, des directeurs de formation, des fondateurs de startups. Les gens qui avaient un budget."

J'ai fait exactement une chose bien : j'ai identifie un marche que personne ne servait (la metacognition IA pour professionnels) et j'ai empile chaque couche de revenue sur la precedente."

## Acte III : Le Premier Echec (Janvier - Mars 2027)

### Centaur OS v0.1 : Le SaaS Premature

Flush de succes, Moison decide en janvier 2027 de construire un SaaS : Centaur OS, une plateforme de diagnostic et de formation metacognitive. L'idee est de transformer son cours en produit logiciel.

"C'etait trop tot," admet-il sans detour. "J'avais 12 000 abonnes YouTube et je me prenais pour un fondateur de SaaS. Le produit etait techniquement bon mais personne n'en voulait au prix que je demandais."

```
+------------------------------------------------------------------+
|  CENTAUR OS v0.1 : AUTOPSIE D'UN ECHEC                          |
|                                                                  |
|  Ce qui a mal tourne :                                            |
|  |-- Prix trop eleve (99 EUR/mois pour un individu)              |
|  |-- Pas de sales pipeline B2B (juste du contenu)               |
|  |-- UI/UX mediocre (construit seul, pas de designer)            |
|  |-- Trop de features, pas assez de focus                        |
|  +-- Le marche n'etait pas pret pour un "SaaS metacognition"    |
|                                                                  |
|  Resultats apres 3 mois :                                        |
|  |-- 47 utilisateurs payants (ironie du chiffre)                 |
|  |-- 4 653 EUR MRR                                               |
|  |-- Churn : 18%/mois (catastrophique)                           |
|  |-- Cout dev + infra : 8 000 EUR/mois                          |
|  +-- Decision : pivoter, pas abandonner                          |
+------------------------------------------------------------------+
```

"Le chiffre qui m'a frappe : 47 utilisateurs. Le meme nombre que mes premieres vues YouTube. L'univers avait un sens de l'humour."

## Acte IV : Le Pivot (Avril - Decembre 2027)

### De SaaS a Plateforme B2B

Au lieu d'abandonner Centaur OS, Moison pivote. Il transforme le produit grand public en plateforme B2B de formation enterprise.

"J'ai realise que les individus n'allaient pas payer 99 euros par mois pour devenir meilleurs avec l'IA. Mais les entreprises allaient payer 10 000 euros pour former 50 managers. C'est le meme produit, la meme technologie, mais un modele completement different."

| Trimestre | Evenement | Revenue mensuel |
|-----------|-----------|----------------|
| Q2 2027 | Pivot B2B. 3 pilotes gratuits. | 45K (cours + communaute) |
| Q3 2027 | 8 entreprises clientes. Certification Qualiopi obtenue. | 80K |
| Q4 2027 | 15 entreprises. Centaur OS B2B en beta payante. | 120K |

### L'Equipe Grandit

"Pendant longtemps j'ai tout fait seul. A 80K par mois, c'etait physiquement impossible de continuer. J'ai embauche mon premier employe en septembre 2027 : une community manager. Puis un developpeur. Puis un commercial B2B. A la fin de 2027, on etait 5."

## Acte VI : La Consolidation (2029)

### Le Near-Burnout

"En mars 2029, j'ai failli craquer. 14 mois sans prendre un vrai jour off. Je dormais 5 heures. Mon HRV etait tombe de 87 a 52. Mon systeme de vie -- ce fameux repository 1-Life que j'avais construit en 2026 -- me hurlait que je derivais."

```
+------------------------------------------------------------------+
|  LE PARADOXE DU SYSTEME                                          |
|                                                                  |
|  Le systeme qu'il avait construit pour gerer sa vie              |
|  lui a SAUVE la vie.                                             |
|                                                                  |
|  Son protocole de tracking detectait :                            |
|  |-- Sommeil en chute (7.5h -> 5h)                               |
|  |-- HRV en chute (87 -> 52)                                    |
|  |-- Exercice abandonne depuis 3 semaines                        |
|  |-- Meditation a zero                                           |
|  +-- Isolation sociale croissante                                 |
|                                                                  |
|  "Le dashboard ne mentait pas. MOI je me mentais."              |

```

Il a pris 3 semaines off en avril 2029. A son retour, il a restructure : embauche un COO, delegue 80% de l'operationnel, et s'est recentre sur la vision et le contenu.

## Epilogue

Gareth Moison n'est pas un genie. Il le dit lui-meme. Ce qu'il est, c'est systematique. Obsessionnellement, methodiquement, incrediblement systematique.

Il a construit un systeme pour penser. Puis un systeme pour apprendre. Puis un systeme pour enseigner. Puis un systeme pour vendre. Chaque couche alimentant la suivante.

En 2026, le monde se precipitait pour utiliser l'IA. Moison, lui, se precipitait pour comprendre comment PENSER avec.

C'est toute la difference.
