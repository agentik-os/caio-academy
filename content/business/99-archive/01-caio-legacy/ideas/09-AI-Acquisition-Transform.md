# Racheter + Transformer: Le Search Fund IA

> Acheter des entreprises traditionnelles ennuyeuses (agence de contenu, cabinet comptable, agence marketing) a 2-5x EBITDA, utiliser l'IA pour 10x l'efficacite, puis revendre ou garder pour le cash flow.

---

## Le Concept

Le "Search Fund IA" est un modele d'acquisition ou tu achetes des entreprises profitables mais sous-optimisees, puis tu utilises l'IA pour transformer radicalement leurs operations. Le delta entre le prix d'achat (base sur les marges actuelles) et la valeur post-transformation (marges multipliees par 3-4x) est ta source de profit.

```
+------------------------------------------------------------------+
|                    LE MODELE ACQUISITION + TRANSFORMATION         |
+------------------------------------------------------------------+
|                                                                    |
|  AVANT (tu achetes)                APRES (tu transformes)         |
|                                                                    |
|  CA: 500K EUR                      CA: 500K EUR (meme)            |
|  Employes: 8                       Employes: 3 + agents IA        |
|  Couts: 425K EUR                   Couts: 200K EUR                |
|  Marge: 15% (75K EUR)              Marge: 60% (300K EUR)          |
|  Valorisation: 3x = 225K EUR       Valorisation: 5x = 1.5M EUR   |
|                                                                    |
|  PRIX D'ACHAT: 225K EUR            VALEUR POST-TRANSFO: 1.5M EUR |
|                                                                    |
|  GAIN NET: 1.5M - 225K = 1.275M EUR                              |
|  (en 12-24 mois de transformation)                                |
+------------------------------------------------------------------+
```

### Pourquoi des Entreprises "Ennuyeuses"

| Critere | Entreprise "Sexy" (SaaS) | Entreprise "Ennuyeuse" (Service) |
|---------|-------------------------|--------------------------------|
| Multiple d'achat | 5-10x ARR | 2-4x EBITDA |
| Prix d'acquisition | 500K-5M EUR | 100K-500K EUR |
| Competition d'achat | Intense (PE, VCs) | Faible (personne ne veut) |
| Cash flow des le jour 1 | Rarement | Toujours |
| Potentiel de transformation IA | Moyen (deja tech) | Enorme (rien n'est automatise) |
| Risque | Eleve (churn, competition) | Faible (clients fideles, previsible) |

## Modele de Monetisation

### 3 Strategies de Sortie

| Strategie | Quand | Gain | Risque |
|-----------|-------|------|--------|
| Flip rapide | 12-18 mois post-achat | 3-6x retour sur investissement | Moyen (marche de revente) |
| Cash cow | Indefini | 300K+/an de cash flow | Faible (revenu stable) |
| Roll-up | 3-5 ans (acheter 3-5 entreprises) | Vendre le groupe 10M+ | Eleve (complexite) |

### Scenario 1: Flip (Buy, Transform, Sell)

```
+------------------------------------------------------------------+
|              SCENARIO FLIP (12-18 MOIS)                           |
+------------------------------------------------------------------+
|                                                                    |
|  Mois 0:  Acheter pour 225K EUR                                   |
|           Financement: 100K cash + 125K seller finance (5 ans)    |
|                                                                    |
|  Mois 1-6:  Transformation IA                                    |
|             Implementer agents IA                                  |
|             Reduire l'equipe (attrition naturelle, pas de licencier)|
|             Former les employes restants                           |
|                                                                    |
|  Mois 7-12: Stabilisation                                        |
|             Prouver les nouvelles marges (3-6 mois de data)       |
|             Documenter les process                                |
|             CA maintenu ou en croissance                          |
|                                                                    |
|  Mois 12-18: Revente                                             |
|              Valorisation: 5x EBITDA = 1.5M EUR                   |
|              - Prix d'achat initial: -225K                         |
|              - Couts de transformation: -30K                       |
|              - Remboursement seller finance: -25K (1 an)          |
|              = PROFIT NET: ~1.2M EUR                               |
+------------------------------------------------------------------+
```

### Scenario 2: Cash Cow (Garder pour le Cash Flow)

| Annee | EBITDA Post-Transfo | Cash Flow Net | Cash Cumule |
|-------|--------------------|----|---|

| 2 | 300K | 250K | 400K |

**Le remboursement du seller financing (125K sur 5 ans = 25K/an) est deja deduit.**

## Risques et Mitigations

| Risque | Probabilite | Impact | Mitigation |
|--------|------------|--------|------------|
| Clients partent apres le rachat | Moyenne | Critique | Transition douce, memes interlocuteurs, meilleur service |
| Employes cles partent | Elevee | Fort | Retention bonus, former avant de reduire, respect |
| Transformation IA echoue | Faible | Fort | Tester sur 1 process d'abord, iterer |
| Problemes caches post-achat | Moyenne | Fort | Due diligence approfondie, clause de garantie |
| Financement refuse | Faible | Critique | Seller financing + epargne personnelle |
| Marche de revente illiquide | Moyenne | Moyen | Cash cow comme plan B (toujours rentable) |
| Risque legal/social (licenciements) | Moyenne | Fort | Attrition naturelle, redeploiement, jamais de licenciement masse |
