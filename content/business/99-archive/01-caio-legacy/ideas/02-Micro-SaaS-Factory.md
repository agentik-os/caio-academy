# Micro-SaaS Factory: Build, Grow, Flip x5

> Construire 3-5 petits outils SaaS propulses par l'IA (5-30K MRR chacun), puis les revendre a 4-8x ARR sur Acquire.com.

---

## Le Concept

La Micro-SaaS Factory est un modele de creation en serie. Au lieu de construire UN gros produit et tout miser dessus, tu construis 3 a 5 petits SaaS en parallele. Chacun cible un probleme de niche tres precis. Tu utilises Claude Code pour builder chaque produit en 2-4 semaines. Tu les fais croitre jusqu'a 5-30K MRR, puis tu les revends sur des marketplaces comme Acquire.com, MicroAcquire, ou FE International.

```
+------------------------------------------------------------------+
|                    LA FACTORY EN ACTION                           |
+------------------------------------------------------------------+
|                                                                    |
|  BUILD          GROW           FLIP                               |
|  (2-4 sem)      (6-12 mois)    (vente)                           |
|                                                                    |
|  [SaaS 1] ----> [15K MRR] ----> VENDU 6x ARR = 1.08M EUR        |
|  [SaaS 2] ----> [8K MRR]  ----> VENDU 5x ARR = 480K EUR         |
|  [SaaS 3] ----> [22K MRR] ----> VENDU 7x ARR = 1.85M EUR        |
|  [SaaS 4] ----> [5K MRR]  ----> GARDE (cashflow)                 |
|  [SaaS 5] ----> [12K MRR] ----> VENDU 5x ARR = 720K EUR         |
|                                                                    |
|  TOTAL EXITS:                    4.13M EUR                        |
|  + CASHFLOW SaaS 4:             60K EUR/an                        |
+------------------------------------------------------------------+
```

|---------|-----------|-----------|
| Temps de build | 2-4 semaines | 6-18 mois |

| Complexite technique | Faible-Moyenne | Elevee |
| Multiplicateur de vente | 4-8x ARR | 3-5x ARR (plus risque) |
| Equipe necessaire | Solo + IA | 3-10 personnes |

## Risques et Mitigations

| Risque | Probabilite | Impact | Mitigation |
|--------|------------|--------|------------|
| Aucun SaaS n'atteint le PMF | Moyenne | Critique | Valider avant de builder (50+ reponses Reddit/IH) |
| Churn eleve (>10%/mois) | Moyenne | Fort | Onboarding solide, support reactif, stickiness features |
| Competition rapide (copie) | Elevee | Moyen | Moats: donnees, integrations, communaute |
| Difficulte a vendre | Faible | Fort | Metriques propres, code documente, pas de dette technique |
| Burn-out: trop de produits | Moyenne | Fort | Max 2 en phase build, automatiser le support |
| API cost explosion | Faible | Moyen | Usage-based pricing, caching, modeles locaux en fallback |
| Acheteur qui se retracte | Faible | Moyen | Multiples offres, broker professionnel |
