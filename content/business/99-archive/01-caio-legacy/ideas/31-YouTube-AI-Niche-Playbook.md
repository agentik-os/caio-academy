## Comprendre l'Algorithme YouTube 2026

### Sources de Trafic (Repartition Moyenne)

| Source | Part du Trafic | Optimisation Cle |
|--------|---------------|------------------|
| Browse Features (accueil) | 48% | CTR thumbnail + titre > 8% |
| Recherche YouTube | 30% | SEO titre, description, tags, chapitres |
| Videos Suggerees | 22% | Retention > 50%, session time eleve |

### Metriques Critiques par Priorite

| Metrique | Seuil Minimum | Objectif | Impact Algo |
|----------|--------------|----------|-------------|
| CTR (Click-Through Rate) | 5% | 8-12% | Tres eleve |
| AVD (Average View Duration) | 40% | 50-65% | Tres eleve |
| Session Time | N/A | Maximiser | Eleve |
| Likes/Comments ratio | 3-5% | 7%+ | Moyen |
| Subscribers gained/video | Varie | Croissant | Moyen |

### Ce qui a Change en 2026

```
AVANT (2023-2024):
- Clickbait fonctionnait
- Frequence > qualite
- Titles sensationnels

MAINTENANT (2026):
- Satisfaction viewer = metrique principale
- "Viewers return rate" pese lourd
- Les Shorts alimentent le long-form (funnel)
- AI content detection (originalite requise)
- Browse Features dominent = thumbnail = ROI
```

## Thumbnails: La Science du Clic

### Formule Gagnante

```
THUMBNAIL = Contraste + Visage/Icone + Texte (max 4 mots) + Declencheur Emotionnel

Exemples:
- Fond bleu vif + visage surpris + "L'IA REMPLACE TOUT"
- Fond noir + logo ChatGPT barre + "C'EST FINI"
- Split screen avant/apres + "SANS CODER"
```

### Regles Techniques

| Parametre | Specification |
|-----------|---------------|
| Resolution | 1280x720 minimum |
| Ratio | 16:9 |
| Taille texte | Lisible sur mobile (70% du trafic) |
| Nombre de mots | 3-4 maximum |
| Couleurs | Contraste fort (complementaires) |
| Visage | Expression exageree OU icone reconnaissable |
| Test | A/B test avec TubeBuddy ou thumbnail.ai |

### Combinaisons de Couleurs Qui Convertissent

| Fond | Texte | CTR moyen observe |
|------|-------|-------------------|
| Bleu fonce | Jaune/blanc | 8-12% |
| Rouge | Blanc | 7-10% |
| Noir | Vert neon | 9-13% |
| Orange | Noir | 8-11% |

## Les 30 Premieres Secondes: Le Hook

### Structure du Hook (Critique)

```
Seconde 0-5:   ACCROCHE (question choc ou statistique)
Seconde 5-15:  PROMESSE (ce que le viewer va obtenir)
Seconde 15-25: PREUVE (credibilite, resultat, experience)
Seconde 25-30: TRANSITION (vers le contenu principal)
```

### Exemples de Hooks

| Type | Exemple | Efficacite |
|------|---------|------------|
| Statistique choc | "86% des employes utilisent l'IA en cachette de leur patron" | Tres elevee |
| Question | "Et si je vous disais que vous pouvez remplacer une equipe de 5 personnes avec un seul prompt?" | Elevee |
| Controverse | "Les cours en ligne sont morts. Completement morts." | Tres elevee |
| Resultat | "Ce systeme me rapporte 12 000 EUR par mois en automatique. Voici comment." | Elevee |
| Prediction | "Dans 18 mois, 40% des agences web auront ferme. Voici pourquoi." | Moyenne-elevee |

## Erreurs Fatales a Eviter

| Erreur | Consequence | Solution |
|--------|------------|----------|
| Niche trop large ("tech") | Algorithme ne sait pas a qui recommander | Niche precise: "IA pour solopreneurs" |
| Pas de hook dans les 5s | 40%+ drop dans les 10 premieres secondes | Script le hook avant tout le reste |
| Thumbnails generiques | CTR < 4% = video invisible | A/B tester, etudier la concurrence |
| Irregularite de publication | Algo penalise les pauses > 2 semaines | Batch recording (2-3 videos d'avance) |
| Ignorer les Shorts | Manquer 30-40% de la decouverte | 3 Shorts/semaine minimum |
| Son mediocre | Viewers quittent en 3 secondes | Micro USB a 80-100 EUR = obligatoire |
