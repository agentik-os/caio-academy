# Coach Communication IA: Parler Mieux, Seduire Mieux

> Analyse tes patterns de communication et t'aide a t'ameliorer.
> Modes : business, dating, networking, negociation.
> Privacy-first : tout peut rester sur ton appareil.

---

## Vision

La communication est la competence la plus sous-estimee et la plus determinante. Un email mal formule perd un client. Un message maladroit tue une attraction. Une negociation mal preparee coute des milliers d'euros. Pourtant, personne n'a de coach de communication en temps reel.

Communication Coach IA comble ce vide. Ce n'est pas un outil qui ecrit A TA PLACE (ca, ChatGPT le fait deja). C'est un coach qui t'ANALYSE, te montre tes faiblesses, et t'aide a progresser. La difference entre un ghostwriter et un coach de tennis.

## Fonctionnalites Principales

### 1. Analyse de Messages

Colle n'importe quel message (email, texto, DM) et recois une analyse :

```
+-- ANALYSE DE MESSAGE --------------------------------------------+
|                                                                   |
|  TON MESSAGE :                                                    |
|  "Salut, je voulais juste savoir si tu avais eu le temps de     |
|   regarder ma proposition ? Pas de rush haha"                    |
|                                                                   |
|  SCORE GLOBAL : 4/10                                              |
|                                                                   |
|  Clarte          [====----------------]  20%  Vague              |
|  Confiance       [====-----------------]  20%  Hesitant           |
|  Persuasion      [=======--------------]  35%  Faible             |
|  Ton             [===========---------]  55%  Trop casual        |
|  Call to Action  [==------------------]  10%  Absent              |
|                                                                   |
|  PROBLEMES DETECTES :                                             |
|  [!] "juste savoir" = minimise ton importance                    |
|  [!] "Pas de rush haha" = signal de faiblesse                   |
|  [!] Pas de deadline ni de call-to-action clair                  |
|  [!] "haha" en contexte pro = manque de serieux                  |
|                                                                   |
|  VERSION AMELIOREE :                                              |
|  "Bonjour Marie, j'espere que ta semaine se passe bien.         |
|   As-tu pu parcourir la proposition ? J'aimerais en discuter    |
|   lors d'un call de 15 minutes cette semaine si possible.       |
|   Mercredi ou jeudi te conviendraient ?"                         |
|                                                                   |
|  POURQUOI C'EST MIEUX :                                           |
|  -> Direct, confiant, propose une action concrete                |
|  -> Donne 2 options (facilite la reponse)                        |
|  -> Pas de minimisation de soi                                   |
+------------------------------------------------------------------+
```

### 2. Simulateur de Conversation

Pratique interactive avant un moment important :

| Scenario | Objectif |
|----------|----------|
| Negociation salariale | S'entrainer a demander + a gerer les objections |
| Premier rendez-vous | Pratiquer l'ecoute active et les questions ouvertes |
| Presentation client | Affiner le pitch et anticiper les questions |
| Confrontation difficile | Gerer le conflit sans agressivite ni passivite |
| Networking event | Pratiquer l'introduction et le small talk |

L'IA joue le role de l'interlocuteur et donne un feedback apres chaque echange.

### 3. Enrichissement du Vocabulaire

Adapte a ton niveau et ton contexte :

| Dimension | Evaluation | Exercices |
|-----------|-----------|-----------|
| Vocabulaire emotionnel | "Tu utilises 12 mots d'emotion, la moyenne est 30" | 3 mots/jour a integrer |
| Precision lexicale | "Tu dis 'truc' et 'chose' 15x/jour" | Alternatives specifiques |
| Power words | "Tes emails manquent de mots d'impact" | Liste de mots persuasifs |
| Connecteurs logiques | "Tes arguments manquent de liaison" | Templates de structure |

### 4. Adaptation Culturelle

| Culture | Ajustements |
|---------|------------|
| France | Plus formel en business, humour autorise en social |
| USA | Plus direct, positif, "can-do attitude" |
| Japon | Indirecte, hierarchie, silence = respect |
| Allemagne | Tres direct, pas de small talk excessif |
| UK | Understatement, politesse indirecte |

### 5. Mode Business

| Feature | Description |
|---------|-------------|
| Email optimizer | Score + rewrite de chaque email pro |
| Meeting prep | Script d'ouverture, questions cles, closing |
| Proposal writing | Structure, ton, persuasion pour propositions |
| Follow-up templates | Relances non-agressives, timing optimal |
| Presentation coach | Structure de pitch, storytelling, anticipation Q&A |

### 6. Mode Dating

| Feature | Description |
|---------|-------------|
| Message review | Analyse de tes messages (trop enthousiaste ? trop distant ?) |
| Conversation practice | Simulateur de premier rendez-vous |
| Profile review | Analyse de profil dating (Hinge, Tinder) |
| Opener generator | Suggestions contextualisees, pas generiques |
| Red flag detector | Analyse des messages recus pour signaux d'alerte |

Positionnement ethique : le but est d'AMELIORER ta communication authentique, pas de manipuler. Le coach t'aide a exprimer qui tu es mieux, pas a pretendre etre quelqu'un d'autre.

### 7. Mode Negociation

```
+-- PREP NEGOCIATION : Augmentation Salariale ---------------------+
|                                                                   |
|  CONTEXTE :                                                       |
|  - Salaire actuel : 55 000 EUR                                  |
|  - Objectif : 65 000 EUR (+18%)                                 |
|  - Anciennete : 3 ans                                            |
|  - Performance : top 10% (evaluations)                           |
|                                                                   |
|  STRATEGIE RECOMMANDEE :                                          |
|  1. Ancrer haut : demander 70K (pour negocier a 65K)            |
|  2. Utiliser les comparables marche (Glassdoor, Welcome)         |
|  3. Lister 3 realisations chiffrees des 12 derniers mois        |
|  4. Preparer le BATNA (meilleure alternative si refus)           |
|                                                                   |
|  OBJECTIONS PROBABLES + REPONSES :                                |
|  "Le budget ne permet pas" -> "Quelles alternatives non-         |
|   salariales pouvons-nous explorer ?"                            |
|  "Pas le bon moment" -> "Quand serait le bon moment et          |
|   quels criteres dois-je remplir ?"                              |
|  "Tu es deja bien paye" -> [Presenter les comparables]          |
|                                                                   |
|  PRATIQUE : Simuler l'entretien (3 rounds recommandes)          |
+------------------------------------------------------------------+
```

### 8. Progression et Tracking

| Metrique | Suivi |
|----------|-------|
| Score moyen par message | Evolution mensuelle |
| Vocabulaire actif | Nombre de mots differents utilises |
| Confiance percue | Score dans les analyses |
| Clarte | Longueur moyenne de phrase, mots vagues |
| Temps de reponse | Patterns de timing dans les conversations |
