## La Methode "IA Socrate"

### Le Principe

```
+------------------------------------------------------------------+
|           TUTEUR CLASSIQUE vs TUTEUR IA SOCRATE                  |
|                                                                  |
|  CLASSIQUE :                                                     |
|  Prof : "La derivee de x^2 est 2x."                             |
|  Eleve : "OK." (memorise, ne comprend pas)                      |
|  Resultat : oublie en 2 semaines                                |
|                                                                  |
|  IA SOCRATE :                                                     |
|  IA : "Si tu traces f(x)=x^2, que remarques-tu sur la pente     |
|        quand x augmente ?"                                       |
|  Eleve : "Elle augmente ?"                                       |
|  IA : "Exact. De combien augmente-t-elle entre x=1 et x=2 ?"   |
|  Eleve : "De 3 ?" (calcule 4-1)                                 |
|  IA : "Bien. Et entre x=2 et x=3 ?"                             |
|  Eleve : "De 5." (calcule 9-4)                                  |
|  IA : "Tu vois un pattern ? 3, 5, 7... Qu'est-ce qui se passe?"|
|  Eleve : "Ca augmente de 2 a chaque fois !"                     |
|  IA : "Et ca, c'est exactement ce qu'on appelle la derivee."    |
|  Resultat : COMPREND, retient pour toujours                     |
|                                                                  |
|  Cle : enseigner par les QUESTIONS, pas par les reponses.       |
|  L'IA pose la bonne question au bon moment.                     |
+------------------------------------------------------------------+
```

### Les Principes Pedagogiques

| Principe | Implementation IA | Etude de reference |
|----------|------------------|-------------------|
| Espacement (spaced repetition) | Rappels automatiques aux intervalles optimaux | Ebbinghaus, Leitner |
| Recall actif | Questions plutot qu'explications | Roediger & Butler 2011 |
| Difficulte desirable | Ajuster la difficulte pour etre juste au-dessus du niveau | Bjork 1994 |
| Feedback immediat | Correction en temps reel avec explication | Hattie & Timperley 2007 |
| Metacognition | L'eleve evalue sa propre comprehension | Flavell 1979 |
| Contextualisation | Exemples du domaine de l'apprenant | Lave & Wenger 1991 |
