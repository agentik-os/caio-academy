## Le Service : Comment Ca Marche

### Le Process Mensuel

```
+------------------------------------------------------------------+
|            WORKFLOW MENSUEL PAR CLIENT                            |
|                                                                  |
|  Etape 1 : Interview mensuelle (30 min, Zoom ou vocal)          |
|  |-- Questions ciblees sur les sujets du mois                   |
|  |-- Capturer les anecdotes, opinions, insights                 |
|  +-- Enregistrement transcrit par IA (Whisper)                   |
|                                                                  |
|  Etape 2 : IA genere les premiers jets (2h)                      |
|  |-- Claude/GPT avec le "voice profile" du client                |
|  |-- 12-15 posts LinkedIn + 4-8 threads Twitter                 |
|  +-- Adapte au style, vocabulaire, ton du client                 |
|                                                                  |
|  Etape 3 : Editeur humain peaufine (3-4h)                       |
|  |-- Verifie l'authenticite de la voix                           |
|  |-- Ajoute les nuances que l'IA manque                          |
|  |-- Optimise pour l'engagement (hooks, structure)               |
|  +-- Selectionne les visuels                                     |
|                                                                  |
|  Etape 4 : Client approuve (15 min)                              |
|  |-- Google Doc partage avec commentaires                        |
|  |-- Client valide ou demande des ajustements                    |
|  +-- Rarement plus de 2 tours de revisions                       |
|                                                                  |
|  Etape 5 : Publication programmee                                |
|  |-- Buffer/Hootsuite ou publication manuelle                    |
|  +-- Reporting mensuel des performances                          |
+------------------------------------------------------------------+
```

## Le "Voice Profile" : La Cle de l'Authenticite

```
+------------------------------------------------------------------+
|           VOICE PROFILE (cree a l'onboarding)                    |
|                                                                  |
|  Elements captures :                                              |
|  |-- Vocabulaire favori (mots recurrents, expressions)           |
|  |-- Ton general (formel, casual, provocateur, pedagogique)      |
|  |-- Longueur de phrases preferee (courtes, longues, mixtes)    |
|  |-- Sujets de predilection                                      |
|  |-- Opinions fortes (controverses qu'ils assument)              |
|  |-- Anecdotes personnelles (repertoire)                         |
|  |-- Style d'argumentation (data-driven, storytelling, etc.)    |
|  +-- Mots/expressions INTERDITS ("leverager", "pivoter", etc.)  |
|                                                                  |
|  Le voice profile est injecte dans chaque prompt IA.            |
|  Resultat : les posts sonnent comme SI le client les avait       |
|  ecrits lui-meme. Car ils sont bases sur SES mots, SES idees.  |
+------------------------------------------------------------------+
```

## Comment Trouver les Clients

### Canaux d'Acquisition

| Canal | Cout | Conversion | Temps avant resultats |
|-------|------|-----------|----------------------|
| LinkedIn outreach directe | 0 EUR | 2-5% | 1-2 mois |
| Ton propre contenu LinkedIn | 0 EUR | Meilleur mais lent | 3-6 mois |
| Referrals (clients existants) | 0 EUR | 30-50% | Apres 3 mois |
| Partenariats (coachs business, CFOs) | 10-20% rev share | 5-15% | 2-4 mois |
| Conferences/meetups business | 500-1000 EUR | 3-8% | 1-3 mois |
| Ads LinkedIn | 30-80 EUR/lead | 1-3% | Immediat |

### Script d'Outreach LinkedIn (Exemple)

```
Message 1 (connection) :
"[Prenom], je vois que vous etes [titre] chez [entreprise].
Votre post sur [sujet] la semaine derniere etait pertinent.
J'aimerais echanger sur [sujet adjacent]."

Message 2 (apres acceptation, J+2) :
"Merci pour la connexion. Question rapide : est-ce que
le personal branding est un sujet strategique pour vous
cette annee ? Je travaille avec des dirigeants sur ce sujet
et j'ai remarque que votre profil a du potentiel inexploite."

Message 3 (si reponse positive) :
"Je fais un audit gratuit de 15 min sur Zoom.
Je vous montre 3 leviers concrets pour doubler votre
visibilite LinkedIn en 30 jours. Pas de vente, juste
de la valeur. Dispo cette semaine ?"
```
