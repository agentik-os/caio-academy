## Comment Negocier ta Valorisation

### Les Leviers

| Levier | Impact sur Valorisation | Comment l'Activer |
|--------|------------------------|-------------------|
| FOMO (multiple term sheets) | +30-50% | Pitcher 5+ VCs en parallele |
| Revenue avec croissance | +50-100% | Meme 5K MRR en croissance vaut plus que 0 |
| Marque fondateur | +20-30% | Audience, contenu, reputation technique |
| Secteur hot (IA agents, souverainete) | +20-40% | Narratif aligne avec les theses VC du moment |
| Traction extraordinaire | +100%+ | Si ton produit est viral, tout change |

### Les Pieges

| Piege | Consequence | Comment Eviter |
|-------|-------------|----------------|
| Valorisation trop haute en seed | Down round en Series A = catastrophe | Accepter 15-20M au lieu de viser 30M |
| Pas de term sheet competing | VC dicte les termes | Toujours pitcher 5+ en parallele |
| Accepter des termes toxiques | Liquidation pref 2x, full ratchet | Avocat startup OBLIGATOIRE |
| Lever trop | Dilution excessive, pression de croissance | Lever 18 mois de runway, pas plus |
| Lever trop peu | Re-lever en 6 mois (distraction) | Minimum 12 mois de runway |

## Formules de Valorisation Utilisees par les VCs

### Methode 1 : Comparable (la plus courante en seed)

```
VALORISATION = ARR x Multiple du secteur

Exemple :
|-- Ton ARR : 120K EUR (10K MRR)
|-- Multiple IA SaaS seed : 15-25x
|-- Valorisation : 1.8M - 3M EUR

MAIS en seed, souvent pas d'ARR. Alors :
|-- Valorisation = "Ce que le marche accepte"
|-- Guide : derniers deals similaires sur Dealroom/Crunchbase
```

### Methode 2 : Scorecard (Payne Method)

| Facteur | Poids | Score (0.5 - 1.5) |
|---------|-------|---------------------|
| Equipe | 30% | 1.0 = moyenne, 1.3 = forte, 0.7 = faible |
| Marche | 25% | 1.0 = 1B, 1.3 = 10B+, 0.7 = < 500M |
| Produit | 15% | 1.0 = MVP, 1.3 = produit lance, 0.7 = idee |
| Concurrence | 10% | 1.0 = plusieurs, 1.3 = peu, 0.7 = beaucoup |
| Marketing | 10% | 1.0 = plan, 1.3 = traction, 0.7 = rien |
| Autres | 10% | Brevets, partenariats, etc. |

```
CALCUL :
1. Valorisation moyenne seed dans ta zone = 12M EUR (reference)
2. Score pondere = somme(poids x score) = ex: 1.15
3. Valorisation ajustee = 12M x 1.15 = 13.8M EUR
```

### Methode 3 : DCF Simplifie (Series A+)

```
ANNEE 3 ARR PROJETE x MULTIPLE SORTIE / RENDEMENT ATTENDU

Exemple :
|-- ARR annee 3 projete : 5M EUR
|-- Multiple exit (IA SaaS) : 10x
|-- Valeur exit : 50M EUR
|-- Rendement attendu VC (10x en 7 ans) : /10
|-- Valorisation aujourd'hui : 5M EUR post-money
|
|-- Mais si le VC ne veut que 20% : 5M / 20% = 25M post-money
+-- Negociation : entre 5M et 25M selon le pouvoir de chaque partie
```
