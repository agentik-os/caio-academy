## Aides Publiques : Le Tresor Cache

```
+-- AIDES PUBLIQUES POUR STARTUPS IA EN FRANCE --------------------+
|                                                                    |
|  BPI FRANCE (Banque Publique d'Investissement)                    |
|  |                                                                |
|  |-- Bourse French Tech          30 000 EUR   Subvention         |
|  |-- Aide a l'Innovation         200-600K     Avance remboursable|
|  |-- Pret d'Amorcage             50-300K      Pret taux zero     |
|  |-- i-Lab (Concours Innovation) 150-600K     Subvention         |
|  |-- France 2030 (IA)            500K-5M      Subvention + pret  |
|  +-- French Tech Seed            250K-500K    Fonds propres      |
|                                                                    |
|  CREDIT IMPOT RECHERCHE (CIR)                                     |
|  |-- 30% des depenses R&D (< 100M EUR)                           |
|  |-- Inclut : salaires chercheurs, sous-traitance, amortissement |
|  |-- Remboursable immediatement pour les PME < 2 ans             |
|  +-- Valeur typique startup IA : 50-200K EUR/an                  |
|                                                                    |
|  STATUT JEI (Jeune Entreprise Innovante)                          |
|  |-- Exoneration charges sociales (50% pendant 8 ans)            |
|  |-- Exoneration impot benefices (100% an 1, 50% an 2)           |
|  |-- Exoneration CFE et CVAE                                     |
|  |-- Conditions : < 8 ans, PME, 15%+ en R&D, independance       |
|  +-- Economie typique : 30-100K EUR/an en charges                |
|                                                                    |
|  FRENCH TECH VISA                                                  |
|  |-- Visa talent 4 ans (fondateur, employe, investisseur)        |
|  |-- Procedure simplifiee (4 semaines)                            |
|  +-- Pour attirer des talents internationaux                      |
|                                                                    |
+--------------------------------------------------------------------+
```

### Calcul Concret des Aides Cumulables

| Aide | Montant | Annee 1 | Annee 2 | Annee 3 |
|------|---------|---------|---------|---------|

| CIR (30% R&D) | Variable | 60 000 | 120 000 | 180 000 |
| JEI (charges sociales) | Variable | 40 000 | 80 000 | 100 000 |
| Aide Innovation BPI | 400K (one-shot) | 400 000 | - | - |
| **TOTAL** | | **530 000** | **200 000** | **280 000** |
