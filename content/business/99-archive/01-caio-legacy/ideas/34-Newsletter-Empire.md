## Format de la Newsletter

### Frequence et Jour d'Envoi

| Parametre | Recommandation | Justification |
|-----------|----------------|---------------|
| Frequence | 2x/semaine | Assez pour l'habitude, pas trop pour la fatigue |
| Jours | Mardi + Jeudi | Meilleurs taux d'ouverture B2B observes |
| Heure | 8h00 CET | Avant le debut de journee, en mode "apprentissage" |
| Longueur | 800-1 200 mots | 3-5 min de lecture (sweet spot attention) |

### Structure de Chaque Edition

```
SUJET (objet email): Curiosite + benefice (max 50 caracteres)
  Exemples: "L'outil IA que Google veut supprimer"
            "3 agents IA, 0 employe, 15K/mois"

INTRO (2-3 lignes): Hook + contexte rapide

SECTION PRINCIPALE (60% du contenu):
  1 insight profond + analyse + implications concretes
  Format: observation -> analyse -> action

3 LIENS CURATIES (30% du contenu):
  - [Outil/Article] + 2 lignes pourquoi c'est pertinent
  - [Outil/Article] + 2 lignes pourquoi c'est pertinent
  - [Outil/Article] + 2 lignes pourquoi c'est pertinent

CTA (10%):
  - Semaine standard: "Reponds a cet email si tu veux [X]"
  - Semaine promo: lien communaute/produit/sponsor
```

### Types d'Editions (Rotation)

| Type | Frequence | Contenu |
|------|-----------|---------|
| Deep Dive | 2x/mois | Analyse approfondie d'un sujet IA |
| Tools Review | 2x/mois | Test detaille d'un outil + verdict |
| Curated | 2x/mois | 5-7 liens commentes (meilleures ressources de la semaine) |
| Strategy | 1x/mois | Business model, monetisation, tactiques |
| Interview/Case Study | 1x/mois | Profil d'un builder IA + lecons |

### Stack de Revenus

| Source | Pre-requis | Revenu Estime |
|--------|-----------|---------------|
| Beehiiv Boost Network | 1K+ subs | 0.5-2 EUR/nouveau sub recommande |
| Sponsors (CPM) | 5K+ subs | 30-80 EUR CPM = 150-400 EUR/edition |
| Affiliation | 1K+ subs | 500-3 000 EUR/mois |
| Tier Premium | 5K+ subs | 7 EUR/mois x 5% = 350 EUR/mois a 1K premium |
| Produits propres | N'importe quand | Variable (communaute, cours, services) |

### Sponsors: Comment les Trouver et les Pricer

```
PHASE 1 (5K-10K subs):
  - Prix: 200-500 EUR/edition
  - Format: 1 paragraphe sponsorise dans la newsletter
  - Ou les trouver: Passionfroot, SponsorGap, DM direct aux outils IA

PHASE 2 (10K-25K subs):
  - Prix: 500-1 500 EUR/edition
  - Format: Section sponsorisee + mention en intro
  - Ou les trouver: Inbound (ils viennent a toi) + plateformes

PHASE 3 (25K-50K subs):
  - Prix: 1 500-4 000 EUR/edition
  - Format: Edition sponsorisee complete (contenu utile autour du sponsor)
  - Ou les trouver: Inbound principalement
```

## Metriques Cles a Suivre

### Dashboard Hebdomadaire

| Metrique | Benchmark Bon | Benchmark Excellent |
|----------|--------------|-------------------|
| Taux d'ouverture | 35-45% | 45-55% |
| Taux de clic | 2-4% | 4-7% |
| Taux de desabonnement | < 0.5%/edition | < 0.2%/edition |
| Growth rate | 5-10%/mois | 10-20%/mois |
| Revenue per subscriber | 0.5-1 EUR/mois | 1-2 EUR/mois |
| Reply rate | 0.1-0.5% | 0.5-2% |

### Signaux d'Alerte

| Signal | Cause Probable | Action |
|--------|---------------|--------|
| Ouverture < 30% | Sujets faibles ou frequence trop haute | A/B test objets + reduire frequence |
| Desabonnement > 1% | Contenu hors sujet ou trop promotionnel | Revenir au contenu pur valeur |
| Clics < 1% | CTA faibles ou liens pas pertinents | Tester nouveaux formats de CTA |
| Croissance stagnante | Canaux d'acquisition satures | Nouveau canal (ads, cross-promo, SEO) |
