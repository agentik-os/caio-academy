# 90-Day Launch Plan -- Du Zero au Premier 50K

*Semaine par semaine. Pas de theorie. Que de l'execution.*

---

## Vue d'Ensemble

| Periode | Mois 1 : FONDATION | Mois 2 : TRACTION | Mois 3 : ACCELERATION |
|---------|--------------------|--------------------|------------------------|
| Semaines | 1-4 | 5-8 | 9-12 |
| Objectif | 1er client | 3-5 clients | Pipeline plein |
| Revenue | 3-5K | 20-35K | 35-50K |
| Content | Lancer la machine | Regularite | Leads inbound |

---

## MOIS 1 : FONDATION

### Semaine 1 -- Infrastructure

**BUSINESS**
- [ ] Creer la SAS (ou auto-entrepreneur pour demarrer vite)
- [ ] Ouvrir compte bancaire pro (Qonto ou Shine)
- [ ] Definir les 3 offres (Audit/Build/CTO) -- DONE via docs
- [ ] Rediger le contrat type (utiliser un template legal)
- [ ] Setup facturation (Stripe ou Pennylane)

**BRAND**
- [ ] Decider du branding (Dafnck Studio evolue ou nouveau nom?)
- [ ] Landing page simple: 1 page, 3 offres, formulaire contact. Build avec Next.js + deploy Vercel (2-3h max)
- [ ] Bio LinkedIn optimisee: "I build products with AI agents"
- [ ] Bio Twitter optimisee: meme angle
- [ ] Photo pro (ou avatar AI de qualite)

**CONTENT**
- [ ] Ecrire le premier thread Twitter (ton histoire)
- [ ] Premier post LinkedIn (story: comment tu en es arrive la)
- [ ] Setup Beehiiv/Substack pour la newsletter
- [ ] Planifier les 4 premieres videos YouTube

### Semaine 2 -- Premier Contenu et Outreach

**CONTENT (lancer la machine)**
- [ ] Video YouTube #1: "I replaced a dev team with AI agents" -- Script + record + edit + publish
- [ ] 3-5 tweets/jour (build in public, insights)
- [ ] 2 posts LinkedIn (story format)
- [ ] Premier article newsletter

**OUTREACH (chercher le 1er client)**
- [ ] Lister 100 fondateurs non-tech sur LinkedIn
- [ ] Envoyer 20 messages personnalises/jour (5 jours = 100)
- [ ] Repondre dans 3-5 communautes Slack/Discord de startups
- [ ] Poster sur Malt/Upwork/Comet (optionnel, pour le flux)
- [ ] Contacter 5 incubateurs/accelerateurs locaux

**PORTFOLIO**
- [ ] Ecrire la case study DentistryGPT (avant/apres, metriques, temoignage si possible)
- [ ] Ecrire la case study Kommu
- [ ] Screenshots + demo video de chaque projet

### Semaine 3 -- Premiers Calls et Ajustement

**SALES**
- [ ] Objectif: 3-5 discovery calls cette semaine
- [ ] Pratiquer le script de discovery call
- [ ] Envoyer 2-3 propositions
- [ ] Follow-up sur les propositions en cours

**CONTENT**
- [ ] Video YouTube #2 (tutorial ou case study)
- [ ] Thread Twitter quotidien
- [ ] 2 posts LinkedIn
- [ ] Newsletter #2

**OUTREACH**
- [ ] 100 nouveaux messages LinkedIn
- [ ] Participer a 1 event/meetup (en ligne ou IRL)
- [ ] Demander des intros aux contacts existants

### Semaine 4 -- Premier Client

**OBJECTIF : CLOSER LE PREMIER CLIENT**

**SALES**
- [ ] Objectif: signer au moins 1 client (meme a tarif reduit). Premier client = case study > revenue
- [ ] Si pas de closing: offrir un audit gratuit/reduit (2K au lieu de 5K). Le but est de PROUVER la methode, pas de gagner de l'argent
- [ ] Begin delivery si client signe

**CONTENT**
- [ ] Video YouTube #3
- [ ] Thread sur "mon premier client AI agents"
- [ ] Newsletter #3

**REVIEW MOIS 1**
- [ ] Combien de leads generes?
- [ ] Combien de calls?
- [ ] Qu'est-ce qui a marche? Qu'est-ce qui n'a pas marche?
- [ ] Ajuster le pitch, le prix, le ciblage si necessaire

## MOIS 2 : TRACTION

### Semaine 5-6 -- Delivery + Momentum

**DELIVERY**
- [ ] Livrer le premier projet (qualite IMPECCABLE)
- [ ] Documenter TOUT le process (pour contenu + case study)
- [ ] Demander un temoignage video au client (30-60 sec)
- [ ] Demander 2-3 referrals au client

**SALES**
- [ ] 100 messages LinkedIn/semaine
- [ ] 5+ discovery calls/semaine
- [ ] Envoyer 3-5 propositions/semaine
- [ ] Objectif: 2 nouveaux clients

**CONTENT**
- [ ] 2 videos YouTube (show the build in progress)
- [ ] Twitter: documenter le delivery en temps reel
- [ ] LinkedIn: case study du premier client
- [ ] Newsletter: behind the scenes

### Semaine 7-8 -- Scale le Pipeline

**SALES**
- [ ] Objectif: 3-5 clients actifs ou signes
- [ ] Augmenter les prix si la demande > capacite
- [ ] Tester les ads retargeting (500 EUR)
- [ ] Automatiser le scheduling (Calendly)

**CONTENT**
- [ ] Video YouTube: case study complete (avant/apres)
- [ ] Thread Twitter viral (les chiffres reels)
- [ ] LinkedIn: "What I learned from my first 5 AI builds"

**SYSTEMS**
- [ ] Creer des templates de proposition
- [ ] Creer un onboarding client automatise
- [ ] Documenter le process de delivery step by step
- [ ] Setup CRM simple (Notion ou Attio)

**REVIEW MOIS 2**
- [ ] Revenue total?
- [ ] Nombre de clients?
- [ ] Meilleur canal d'acquisition?
- [ ] Pipeline pour mois 3?

## MOIS 3 : ACCELERATION

### Semaine 9-10 -- Professionnaliser

**BRAND**
- [ ] Site web complet (pas juste landing): portfolio, about, blog
- [ ] 3+ case studies publiees
- [ ] Page temoignages avec videos
- [ ] Media kit pour events/podcasts

**SALES**
- [ ] Inbound > Outbound (objectif)
- [ ] Process de qualification automatise
- [ ] Augmenter les prix de 20-30%
- [ ] Selectionner les meilleurs clients (dire NON)

**CONTENT**
- [ ] 2 videos YouTube/semaine (routine installee)
- [ ] Newsletter: 500+ subs objectif
- [ ] Twitter: 1000+ followers objectif
- [ ] Premier guest podcast ou interview

### Semaine 11-12 -- Optimiser et Scaler

**SCALE**
- [ ] Identifier le canal #1 (celui qui genere le plus de leads)
- [ ] Doubler l'effort sur ce canal
- [ ] Reduire/eliminer ce qui ne marche pas
- [ ] Lancer les ads sur le canal #1 (1-2K EUR/mois)

**REVENUE PASSIF (debut)**
- [ ] Commencer a ecrire le "AI Development Playbook"
- [ ] Creer 1-2 templates vendables (landing page, SaaS starter)
- [ ] Reflechir au cours en ligne (pour Mois 6+)
