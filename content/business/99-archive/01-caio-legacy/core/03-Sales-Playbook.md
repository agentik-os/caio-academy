# Sales Playbook -- Process de Vente

*Du premier contact a la signature. Chaque etape optimisee pour convertir.*

---

## Vue d'Ensemble du Funnel

| Etape | Taux | Action cle |
|-------|------|------------|
| Prospects | 100% | Contenu gratuit, outreach, referrals |
| Leads Qualifies | 20-30% | Discovery call 30min |
| Proposition Envoyee | 15-20% | Proposition 3 offres + follow-up |
| Negociation | 10-15% | Objections handling + closing |
| Client Signe | 8-12% | Onboarding + Delivery |
| Referral | 50% des clients satisfaits | Temoignage + introductions |

**Objectif :** 3-5 clients/mois avec un taux de closing de 30-40% sur les calls.

---

## Phase 1 : Generation de Leads

### Canal 1 : LinkedIn Outreach (principal B2B)

**Cible :** Fondateurs non-tech, CEOs de PME, Head of Innovation

**Template message froid :**

> [PRENOM], j'ai vu que vous travaillez sur [PROJET/SECTEUR].
>
> Je suis curieux : avez-vous explore comment les agents IA peuvent accelerer votre developpement produit ?
>
> Je viens de construire un MVP complet en 3 semaines avec mes agents pour un client dans [SECTEUR SIMILAIRE]. Avant, ca aurait pris 4 mois.
>
> Si ca vous intrigue, je serais ravi d'en discuter 15 minutes.
>
> -- Gareth

**Volume :** 10-20 messages/jour, 5 jours/semaine = 50-100/semaine

**Taux de reponse attendu :** 5-10% = 3-10 conversations/semaine

### Canal 2 : Contenu Inbound

Les leads viennent a toi via :
- YouTube : videos qui montrent le process
- Twitter : threads "build in public"
- LinkedIn : articles "how I built X with AI"
- Newsletter : nurturing long terme

### Canal 3 : Referrals

Apres chaque projet reussi :
- Demander un temoignage video (30 secondes)
- Demander 2-3 introductions
- Offrir 10% commission sur les referrals qui signent

### Canal 4 : Communautes & Events

- Participer aux events startup (Station F, BPI, incubateurs)
- Poster dans les Slack/Discord de fondateurs
- Meetups tech locaux
- Demo days de startups (ils cherchent tous un CTO)

## Phase 2 : Discovery Call (30 minutes)

### Structure du Call

**Minutes 0-5 : Rapport**
- "Parlez-moi de vous et de votre projet"
- Ecouter ACTIVEMENT
- Montrer que tu comprends leur monde

**Minutes 5-15 : Diagnostic**
- "Quel est votre plus gros blocage tech aujourd'hui?"
- "Combien ca vous coute de ne PAS avoir cette solution?"
- "Vous avez deja essaye quoi?"
- "Quel est votre timeline ideal?"
- "Quel budget avez-vous envisage?"

**Minutes 15-25 : Solution**
- "Voici comment je travaille" (expliquer le process AI)
- Montrer 1-2 case studies pertinentes
- "Pour votre projet, je vois [APPROCHE]"
- Donner une estimation de timeline

**Minutes 25-30 : Next Steps**
- "Je vais vous envoyer une proposition avec 3 options"
- "Vous la recevez sous 48h"
- "On planifie un call de suivi dans 5 jours?"

### Questions Cles a Poser

**Pour qualifier le lead :**

| Question | Ce que tu cherches |
|----------|--------------------|
| "Quel est votre budget pour ce projet?" | Capacite financiere |
| "Qui decide du budget?" | Decision maker |
| "Quand voulez-vous lancer?" | Urgence |
| "Avez-vous deja un produit/prototype?" | Scope |
| "Combien d'utilisateurs visez-vous au launch?" | Ambition realiste |

**Pour creer l'urgence :**

| Question | Effet |
|----------|-------|
| "Combien vous coute chaque mois SANS cette solution?" | Pain quantifie |
| "Que se passe-t-il si vous ne lancez pas avant [DATE]?" | Peur de manquer |
| "Vos concurrents utilisent-ils deja l'IA?" | Pression competitive |

### Red Flags (quitter le call poliment)

| Red Flag | Action |
|----------|--------|
| "Pas de budget, juste equity" | "Mes offres commencent a 3K, est-ce realiste?" |
| "On veut voir d'abord" | "Je comprends, mon audit a 3K est fait pour ca" |
| "On a deja 3 devs mais ils sont nuls" | Management problem, pas tech problem |
| "On veut copier [concurrent]" | Pas de vision propre |
| "C'est ultra urgent, on lance demain" | Chaos garanti |

## Phase 3 : Proposition

### Format de la Proposition (1 page)

**En-tete :**

| Element | Detail |
|---------|--------|
| Projet | [NOM DU PROJET] |
| Client | [NOM DU CLIENT] |
| Date | [DATE] |
| Validite | 14 jours |

**Section "Votre Situation" :** 2-3 phrases resumant leur probleme et objectif.

**Section "Ma Recommandation" :** 1 paragraphe expliquant pourquoi l'approche AI-first est ideale pour eux.

**Section "3 Options" :**

**Option A : Audit "Clarity" -- [PRIX] EUR**
- Ce qui est inclus
- Livrable
- Timeline

**Option B : Build "Velocity" -- [PRIX] EUR [RECOMMANDE]**
- Tout de A +
- Ce qui est inclus en plus
- Livrable
- Timeline

**Option C : CTO "Partnership" -- [PRIX] EUR/mois + [X]% equity**
- Tout de B +
- Ce qui est inclus en plus
- Livrable
- Timeline

**Section "Next Step" :** Choisissez l'option qui vous convient et repondez a cet email. Je vous envoie le contrat dans les 24h.

### Delai d'envoi

Envoyer la proposition **sous 24-48h** apres le call. Plus c'est rapide, plus le taux de closing est eleve.

## Phase 4 : Closing & Objections

### Objections Frequentes et Reponses

**"C'est trop cher"**

"Je comprends. Comparons : une equipe traditionnelle couterait [3-5x mon prix] et prendrait [3-5x le temps]. Mon prix reflete la valeur du resultat, pas le nombre d'heures. Et avec l'option paiement en 3x, ca represente [X] EUR/mois."

**"Et si ca ne marche pas?"**

"J'ai livre [X] projets avec cette methode. Voici les case studies. De plus, avec l'offre Audit, vous pouvez tester pour [3-7K] avant de vous engager sur un build complet."

**"L'IA ca fait de la merde"**

"L'IA seule, peut-etre. L'IA avec 2 ans d'expertise et des process testes sur [X] projets, ca donne des resultats. Voulez-vous voir le code du dernier projet que j'ai livre?"

**"On va y reflechir"**

"Bien sur. Pour vous aider dans votre reflexion : a combien estimez-vous chaque semaine de retard sur votre lancement? Ma proposition est valable 14 jours."

**"Mon dev peut le faire"**

"Votre dev est certainement competent. La question est : en combien de temps? Si je peux livrer en 4 semaines ce qu'il ferait en 4 mois, ca libere votre dev pour d'autres priorites."

**"Je veux juste l'equity deal, pas de cash"**

"Je comprends l'attrait. Mais mon modele est cash + equity pour aligner nos interets. Le cash garantit mon engagement immediat, l'equity garantit mon engagement long terme. C'est un win-win."

## Phase 5 : Onboarding Client

### Jour 1-2 : Setup

- Contrat signe, premier paiement recu
- Call kickoff 45min (deep dive projet)
- Creation repo + projet + acces
- Setup Slack/Discord channel dedie
- Partage du planning semaine par semaine

### Communication Ongoing

| Frequence | Format | Contenu |
|-----------|--------|---------|
| Quotidien | Message async (Slack) | Update rapide, questions |
| Hebdomadaire | Loom video 5min | Demo du progres |
| Bi-hebdomadaire | Call 30min (optionnel) | Review + feedback |
| Livraison | Call 45min | Demo finale + handover |

### Livraison Finale

- Demo live du produit
- Documentation technique
- Acces admin complets transferes
- Guide de maintenance
- Formation utilisation (video enregistree)
- Facture finale envoyee
- Demande de temoignage
- Demande de referrals
