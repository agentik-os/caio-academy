## Psychologie Derriere Chaque Prix

### Offre 1 : Audit "Clarity" (3K-7K)

**Role psychologique :** Point d'entree. Reduit le risque percu. Le client "teste" sans gros engagement.

**Pourquoi ca marche :**
- 3-7K est un montant "facile" pour un fondateur funded
- Le livrable est tangible (document + setup)
- Ca cree un BESOIN pour l'offre 2 ("maintenant je vois tout ce qui est possible")
- 80% des clients Audit upgraderont vers Build

**Ancrage :** "Un consultant McKinsey coute 2K/jour. Moi je vous donne un audit complet + setup pour le prix d'une journee et demie de consulting traditionnel."

### Offre 2 : Build "Velocity" (10K-30K)

**Role psychologique :** L'offre principale. Celle que tu veux vendre le plus.

**Pourquoi ca marche :**
- Ancrage vs embauche : "15K vs 150K pour la meme chose"
- Ancrage vs agence : "20K vs 100K pour le meme resultat"
- Le client voit le ROI immediatement
- Delai court (3-6 semaines) = dopamine rapide

**Grille de prix detaillee :**

| Complexite | Duree | Prix | Equity optionnel |
|-----------|-------|------|------------------|
| Landing + CRM simple | 1-2 sem | 5-10K | 0% |
| MVP SaaS basique | 3-4 sem | 10-18K | 0-3% |
| MVP SaaS complet | 4-6 sem | 15-25K | 3-5% |
| Produit complexe | 6-8 sem | 25-35K | 5-8% |
| App mobile + web | 6-10 sem | 30-50K | 5-10% |

### Offre 3 : CTO "Partnership" (3-5K/mois + equity)

**Role psychologique :** L'offre premium. Cree un engagement long terme et des revenus recurrents.

**Pourquoi ca marche :**
- Le fondateur a un "CTO" sans le cout d'un CTO
- L'equity aligne les interets
- Le retainer est previsible pour les deux parties
- 3-5K/mois est RIEN compare a 120K/an pour un CTO

**Structure equity :**

| Stade startup | Retainer | Equity | Vesting |
|--------------|----------|--------|---------|
| Pre-revenue | 2-3K/mois | 12-15% | 3 ans, cliff 6 mois |
| Revenue < 10K/mois | 3-4K/mois | 8-12% | 3 ans, cliff 6 mois |
| Revenue > 10K/mois | 4-6K/mois | 5-8% | 2 ans, cliff 3 mois |
| Revenue > 50K/mois | 6-10K/mois | 3-5% | 2 ans |

## Techniques d'Ancrage

### Ancrage par comparaison

Dans chaque proposition, inclure une section "Alternatives" :

**Option A : Embaucher une equipe**

| Element | Detail |
|---------|--------|
| Composition | 2 devs + 1 PM |
| Cout | 180K EUR/an |
| Temps avant resultat | 6-12 mois |
| Risque | Turnover, mauvais recrutement |

**Option B : Agence traditionnelle**

| Element | Detail |
|---------|--------|
| Type | MVP standard |
| Cout | 80K-150K EUR |
| Temps | 4-8 mois |
| Risque | Depassement budget, retards |

**Option C : Freelance**

| Element | Detail |
|---------|--------|
| Profil | Dev senior |
| Cout | 500-800 EUR/jour = 50-80K pour un MVP |
| Temps | 3-6 mois |
| Risque | Disponibilite, qualite variable |

**Ma Proposition : AI-First Development**

| Element | Detail |
|---------|--------|
| Type | MVP complet |
| Cout | 15-25K EUR |
| Temps | 3-6 semaines |
| Risque | Garanti par portfolio + methode prouvee |

### Ancrage par valeur creee

**Votre investissement :** 20K EUR

**Valeur creee :**
- Produit qui vaudrait 80-150K en dev traditionnel
- 6 mois de temps gagne (valeur incalculable)
- Stack technique moderne, maintenable
- Documentation complete
- Tests automatises (MANIAC)
- Deploy production ready

**ROI : 4x-7x des le premier jour.**

## Pricing Mistakes a Eviter

| Erreur | Pourquoi c'est une erreur | Solution |
|--------|--------------------------|----------|
| Prix trop bas | Attire les mauvais clients, devalue ton expertise | Minimum 3K pour le consulting |
| Prix a l'heure | Penalise ta vitesse (les agents vont VITE) | Toujours au forfait |
| Pas d'equity | Tu laisses l'upside sur la table | Toujours proposer equity en option |
| Pas de structure 3 offres | Le client ne sait pas comparer | Toujours 3 options |
| Negocier sur le prix | Signals faiblesse | Negocie sur le scope, jamais le prix |
| Pas de terms clairs | Disputes futures | Contrat ecrit TOUJOURS |

## Script de Prix (quand le client demande "combien?")

"Mes projets commencent generalement a partir de 10K pour un MVP et montent a 30-50K pour les produits plus complexes. Mais le plus important c'est de comprendre votre vision d'abord. Je vous propose un call de 30 minutes gratuit pour evaluer votre projet. Si ca matche, je vous envoie une proposition detaillee avec 3 options adaptees a votre budget et vos ambitions."

**Ne jamais donner le prix exact avant de comprendre le projet.**
