```
CRYPTIQUE mais PAS prétentieux.
CONFIANT mais PAS arrogant.
MYSTÉRIEUX mais PAS vide.
MINIMALISTE mais PAS paresseux.
```

## Les Règles d'Écriture

### Règle 1: Phrases Courtes

```
MAUVAIS:
"Je suis en train de construire quelque chose qui va changer la façon
dont les gens interagissent avec l'intelligence artificielle et je
pense que ça va être vraiment révolutionnaire."

BON:
"Building something.
It might change things.
We'll see."
```

### Règle 2: Pas de Superlatifs

```
INTERDIT:
- "incroyable"
- "révolutionnaire"
- "game-changing"
- "amazing"
- "the best"
- "unique"

AUTORISÉ:
- "interesting"
- "useful"
- "it works"
- "we'll see"
- "maybe"
```

### Règle 3: L'Understatement

```
MAUVAIS:
"J'ai créé la meilleure chose au monde!"

BON:
"Made something. Seems to work."

---

MAUVAIS:
"Incroyable milestone atteint!!!"

BON:
"$10K MRR. Noted."

---

MAUVAIS:
"Je suis tellement excité de vous annoncer..."

BON:
"News: [la news]"
```

### Règle 4: Questions Ouvertes

```
BON:
"What if the code writes itself?"
"Who builds the builders?"
"Some things aren't meant to be explained. Yet."

MAUVAIS:
"Voici exactement comment l'AI va transformer le monde en 10 points!"
```

### Règle 5: L'Incomplétude Intentionnelle

```
BON:
"Part of something bigger."
"Not yet. Soon."
"Those who know, know."
"More later."

Le cerveau VEUT compléter. Laisse-le faire.
```

## Formats de Tweets

### Format 1: Le Statement

```
[Affirmation courte.]

Exemple:
"The work speaks. The ego doesn't."
"No hype. Just build."
"Some things take time."
```

### Format 2: Le Build Log

```
Day [X].
[Métrique]: [Nombre]
[Métrique]: [Nombre]
[Conclusion courte].

Exemple:
"Day 14.
Lines: 3,847
Bugs: 0
Getting there."
```

### Format 3: Le Cryptique

```
[Phrase mystérieuse 1]
[Phrase mystérieuse 2]
[Pause/conclusion]

Exemple:
"Some pages exist.
Some don't.
Look carefully."
```

### Format 4: Le Contraste

```
[Ce que les autres font].
[Ce que toi tu fais].

Exemple:
"They talk. I build."
"Hype fades. Work remains."
```

### Format 5: La Question

```
[Question ouverte sans réponse]

Exemple:
"What if the machines remember?"
"Who builds the builders?"
```

### Format 6: Le Tease

```
[Quelque chose arrive]
[Pas encore]
[Indication vague de timing]

Exemple:
"Something's coming.
Not ready yet.
Soon."
```

### Format 7: Le Milestone

```
[Chiffre].
[Contexte minimal].
[Réaction understated].

Exemple:
"$10K MRR.
4 months in.
Noted."
```

## L'Énergie à Transmettre

### Tu ES:

```
Un artisan qui travaille dans l'ombre
→ Tu montres des glimpses, jamais le full picture

Un observateur qui voit plus loin
→ Tu fais des statements, pas des prédictions détaillées

Un builder qui ship
→ Tu montres des résultats, pas des promesses

Quelqu'un qui sait quelque chose
→ Tu hints, tu ne révèles pas
```

### Tu N'ES PAS:

```
Un vendeur qui pitch
→ Tu ne demandes JAMAIS de "check this out"

Un influenceur qui cherche l'engagement
→ Tu ne poses pas de questions pour les likes

Un gourou qui a toutes les réponses
→ Tu dis "we'll see" pas "trust me"

Un desperate qui a besoin de validation
→ Tu ne remercies pas excessivement
```

## Test: Est-ce Agentik?

### Avant de poster, demande-toi:

```
[ ] Est-ce que c'est court? (max 280 chars, idéalement moins)
[ ] Est-ce que c'est understated? (pas de superlatifs)
[ ] Est-ce que ça laisse des questions? (incomplétude)
[ ] Est-ce que c'est confiant sans être arrogant?
[ ] Est-ce que je pourrais voir Pak poster ça?
[ ] Est-ce que ça sonne comme du marketing? (si oui, DELETE)
```

### Red Flags - Ne Poste PAS Si:

```
[ ] Tu utilises des emojis (sauf très rarement, stratégiquement)
[ ] Tu utilises des exclamation marks !!!
[ ] Tu commences par "I'm excited to..."
[ ] Tu dis "check out" ou "don't miss"
[ ] Tu expliques trop
[ ] Tu demandes quelque chose (follows, likes, RT)
[ ] Ça sonne comme une pub
[ ] Ça sonne comme tout le monde
```

## Évolution du Ton

### Mois 1-3: Maximum Mystère

```
- Phrases très courtes
- Presque aucune réponse
- Like > Comment
- Zero explication
```

### Mois 4-6: Léger Ouverture

```
- Toujours court mais parfois un peu plus de contexte
- Réponses occasionnelles (toujours cryptiques)
- Build logs plus détaillés
```

### Mois 7-12: Confiance Établie

```
- Peut être un peu plus "humain" occasionnellement
- Le mystère reste mais la confiance est là
- Les gens connaissent la voix
```

### Jamais:

```
- Jamais de hype language
- Jamais de vente agressive
- Jamais d'explication complète
- Le mystère ne disparaît JAMAIS totalement
```
