## Le Test en Detail

### Section 1: Raisonnement Logique

```
EXEMPLES DE QUESTIONS:

1. PATTERN RECOGNITION
   "Quelle est la prochaine figure dans cette sequence?"
   [Serie de formes geometriques complexes]

2. LOGICAL DEDUCTION
   "Si A implique B, et B implique C, mais pas D..."
   [Puzzle logique multi-etapes]

3. NUMERICAL REASONING
   "Trouvez la regle qui genere cette sequence:"
   2, 5, 11, 23, 47, ?

SCORING:
- Reponse correcte: +10 points
- Temps bonus: +1-5 points
- Difficulte adaptive
```

### Section 2: Problem Solving Novel

```
PRINCIPE:
Problemes UNIQUES generes par AI pour chaque test.
Impossible de preparer ou memoriser.

EXEMPLES:

1. "Une startup a 3 produits. Le produit A cannibalise B
    de 15% mais augmente C de 22%. B represente 40% du CA.
    Faut-il tuer A? Montrez votre raisonnement."

2. "Vous avez 100 serveurs et un budget de $50K/mois.
    Optimisez cette architecture pour maximiser uptime
    tout en minimisant latence. Contraintes: [...]"

3. "Un gouvernement veut implementer un UBI.
    Identifiez 5 effets de second ordre que personne
    n'a encore mentionnes publiquement."

SCORING:
- Process > Answer
- Multiple valid solutions
- Human graders + AI assist
```

### Section 3: AI Collaboration

```
SETUP:
Le candidat a acces a Claude/GPT pendant cette section.
On mesure COMMENT il utilise l'AI.

TASK EXAMPLE:
"Analysez ce dataset de 10,000 lignes et trouvez
les 3 insights les plus actionables pour un CMO."

WHAT WE MEASURE:
1. PROMPT QUALITY
   - Clarity
   - Specificity
   - Context provided

2. ITERATION SKILL
   - How they refine prompts
   - How they handle errors
   - How they dig deeper

3. VERIFICATION
   - Do they fact-check AI?
   - Do they ask for sources?
   - Critical thinking

4. SYNTHESIS
   - Can they combine AI output with own thinking?
   - Do they add value beyond AI?

SCORING:
- Prompt quality: 0-100
- Iteration: 0-100
- Verification: 0-100
- Synthesis: 0-100
- Total: 0-400 (normalized to 250)
```

### Section 4: Creativity & Synthesis

```
TASKS:

1. DIVERGENT THINKING
   "Listez 20 usages non-conventionnels pour [objet random]"
   Mesure: Quantite + Originalite + Faisabilite

2. CONCEPT COMBINATION
   "Combinez [concept A] et [concept B] pour creer
    quelque chose de nouveau et utile."
   Ex: "Combinez la blockchain et l'education prescolaire"

3. SCENARIO PLANNING
   "L'AI devient 10x moins chere demain.
    Decrivez 5 business qui n'existent pas encore
    mais qui deviendraient inevitables."

SCORING:
- Originalite: 0-100
- Faisabilite: 0-100
- Depth: 0-100
- Total: 0-300 (normalized to 250)
```

## Stack Technique

### Test Platform

```
ARCHITECTURE:

FRONTEND:
+-- Next.js (test interface)
+-- Proctoring SDK (screen recording, webcam)
+-- Timer + auto-submit
+-- Responsive (desktop required)

BACKEND:
+-- Node.js + Convex
+-- Question bank (10,000+ questions)
+-- Adaptive difficulty algorithm
+-- AI grading for open-ended

PROCTORING:
+-- Webcam monitoring (AI)
+-- Screen recording
+-- Tab switching detection
+-- Copy/paste blocking
+-- Anomaly detection

AI GRADING:
+-- Claude API for open-ended evaluation
+-- Rubric-based scoring
+-- Human review for edge cases
+-- Calibration continuelle

SECURITY:
+-- Unique questions per test
+-- Randomized order
+-- Time limits per section
+-- IP tracking
+-- Device fingerprinting
```

### Blockchain Verification

```
CERTIFICATE ON-CHAIN:

1. After test completion:
   - Hash: (user_id + score + timestamp + test_id)
   - Store on Polygon
   - Cost: ~$0.01

2. Verification:
   - QR code on certificate
   - Links to verification page
   - Shows: Score, Date, Hash, Block

3. Immutable record:
   - Can't be faked
   - Can't be changed
   - Permanent proof
```
