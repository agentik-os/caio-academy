## Tech Stack

### Platform Architecture

```
FRONTEND:
+-- Next.js (speed critical)
+-- Tailwind + Framer Motion
+-- Real-time countdown
+-- Instant checkout

BACKEND:
+-- Convex (real-time inventory)
+-- Edge functions (low latency)
+-- Queue system (fair access)

PAYMENTS:
+-- Stripe (instant)
+-- Crypto option (ETH, USDC)

INVENTORY:
+-- Real-time counter
+-- Lock on add-to-cart (30 sec)
+-- Release if not purchased

ANTI-BOT:
+-- Cloudflare Turnstile
+-- Browser fingerprinting
+-- Rate limiting
+-- Queue randomization
```

### Drop Execution

```
DROP FLOW:

T-0:00:00 → Page unlocks
T-0:00:01 → Queue forms
T-0:00:02 → First checkouts start
T-0:00:30 → 50% sold (typical)
T-0:02:00 → Sold out (target)

TECHNICAL CHALLENGES:
- Traffic spike (10x normal)
- Bot prevention
- Fair access
- Payment failures

SOLUTIONS:
- CDN caching
- Serverless scale
- Queue system
- Retry logic
```
