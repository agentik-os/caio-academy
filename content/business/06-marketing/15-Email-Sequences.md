### A. Welcome Sequence (5-7 Emails)

**Objectif:** Creer une relation, livrer la promesse, etablir l'autorite.

```
TIMING: 1 email tous les 1-2 jours pendant 7-10 jours
TAUX D'OUVERTURE CIBLE: 50%+ (les welcome emails ont le plus haut taux)
```

| Email | Jour | Sujet | Contenu | CTA |
|-------|------|-------|---------|-----|
| W1 | J+0 | Livraison lead magnet | Lien de telechargement + "Qui suis-je en 3 lignes" | Telecharger |
| W2 | J+1 | Ton histoire | Pourquoi tu fais ce que tu fais, parcours personnel | Repondre a l'email |
| W3 | J+2 | Quick win | Un conseil actionnable immediat, valeur pure | Appliquer le conseil |
| W4 | J+4 | Preuve sociale | Resultat d'un projet, temoignage, stats | Voir le projet |
| W5 | J+5 | Ta philosophie | Ce en quoi tu crois (build in public, AI, etc.) | Rejoindre la communaute |
| W6 | J+7 | Segmentation | "Qu'est-ce qui t'interesse le plus?" 2-3 choix | Cliquer pour choisir |
| W7 | J+9 | Soft pitch | "Au fait, j'ai cree [produit] pour les gens comme toi" | Decouvrir le produit |

**Structure Detaillee Email W1 (Bienvenue):**

```
SUJET: Voici ton [Lead Magnet] (+ un bonus)

---

Salut {prenom},

[1 phrase de bienvenue chaleureuse]

Voici ton [lead magnet]:
--> [LIEN]

[2 phrases sur ce que tu vas recevoir dans cette newsletter]

Le deal:
- Je t'envoie [frequence] un email avec [type de contenu]
- Pas de spam. Pas de pitch constant. De la valeur.
- Tu peux repondre a chaque email, je lis tout.

[1 phrase humaine/personnelle]

A bientot,
Gareth

P.S. -- Reponds a cet email avec un mot sur ce que
tu construis. Je lis chaque reponse.
```

### B. Nurture Sequence (Continue)

**Objectif:** Apporter de la valeur regulierement. Construire la confiance sur le long terme.

```
LA REGLE 80/20:
+------------------------------------------------------+
|                                                       |
|  80% de tes emails = VALEUR PURE                     |
|  - Tutoriels                                          |
|  - Insights business                                  |
|  - Outils decouverts                                  |
|  - Lecons apprises                                    |
|  - Behind the scenes                                  |
|                                                       |
|  20% de tes emails = PROMOTION                        |
|  - Mention de ton produit                             |
|  - Offre speciale                                     |
|  - Temoignage client + lien                           |
|  - "J'ai cree ca pour toi"                           |
|                                                       |
|  Si tu donnes de la valeur 80% du temps,             |
|  les gens VEULENT acheter tes 20% de promo.          |
|                                                       |
+------------------------------------------------------+
```

**Types d'Emails Nurture:**

| Type | Frequence | Contenu | Exemple |
|------|-----------|---------|---------|
| Newsletter hebdo | 1x/semaine | 3-5 insights de la semaine | "Cette semaine j'ai decouvert..." |
| Tutorial | 2x/mois | Guide etape par etape | "Comment automatiser [X] avec AI" |
| Story email | 2x/mois | Lecon via storytelling | "J'ai perdu un client. Voici pourquoi." |
| Curation | 1x/mois | Meilleurs outils/ressources | "5 outils que j'utilise ce mois" |
| Behind the scenes | 2x/mois | Revenus, stats, process | "Janvier: 3,400 EUR MRR. Breakdown:" |
| Opinion piece | 1x/mois | Take controversee ou vision | "L'IA ne va pas remplacer les devs." |

```
|  PHASE 3: VALEUR (2 emails)                          |

|                                                       |

|                                                       |
+------------------------------------------------------+
```

|-------|--------|-------|-------|
| A1 | +1h | "Tu as oublie quelque chose?" | Rappel simple |

### E. Onboarding Sequence (Post-Achat)

**Objectif:** Activer le client, reduire le churn, creer un ambassadeur.

| Email | Timing | Contenu | Objectif |
|-------|--------|---------|----------|
| O1 | Immediat | Confirmation + acces + premier pas | Activation |
| O2 | +1 jour | Quick start guide + video tuto | Premiere victoire |
| O3 | +3 jours | "Comment vas-tu? Besoin d'aide?" | Support proactif |
| O4 | +7 jours | Tips avances + cas d'usage | Valeur avancee |
| O5 | +14 jours | "Partage ton experience" + request review | Preuve sociale |
| O6 | +30 jours | Cross-sell / upsell | Revenue additionnel |

### F. Re-engagement Sequence

**Objectif:** Reveiller les abonnes inactifs (pas d'ouverture depuis 90+ jours).

```
STRATEGIE RE-ENGAGEMENT:
+------------------------------------------------------+
|                                                       |
|  Identifier les inactifs: pas d'ouverture 90+ jours |
|                                                       |
|  Email R1: "Tu es toujours la?"                      |
|  --> Sujet court, intrigant                           |
|  --> Rappeler pourquoi ils se sont inscrits           |
|                                                       |
|  Email R2: "Ce que tu as rate"                        |
|  --> Best-of du contenu recent                        |
|  --> Quick win immediat                               |
|                                                       |
|  Email R3: "Derniere chance avant suppression"        |
|  --> "Si je n'ai pas de tes nouvelles, je te retire" |
|  --> Bouton: "Je veux rester"                         |
|  --> FONCTIONNE: les gens detestent perdre un acces  |
|                                                       |
|  Post-sequence:                                       |
|  - Ceux qui ouvrent --> retour en nurture            |
|  - Ceux qui ignorent --> SUPPRIMER de la liste       |
|  --> Une liste propre = meilleure deliverability     |
|                                                       |
+------------------------------------------------------+
```

### Structure d'un Email Parfait

```
+-- STRUCTURE EMAIL ----------------------------------------+
|                                                           |
|  SUBJECT LINE                                            |
|  --> La promesse (max 50 caracteres)                     |
|                                                           |
|  PREVIEW TEXT                                             |
|  --> Complete le subject (max 90 caracteres)             |
|  --> Apparait apres le sujet dans l'inbox                |
|                                                           |
|  HOOK (premiere ligne)                                    |
|  --> Arreter le scroll dans l'inbox                      |
|  --> Question, stat choc, ou declaration bold            |
|                                                           |
|  STORY (corps de l'email)                                |
|  --> Raconter, pas expliquer                             |
|  --> Paragraphes courts (2-3 lignes max)                 |
|  --> Langage conversationnel                             |
|  --> Comme si tu ecrivais a UN ami                       |
|                                                           |
|  LESSON (le takeaway)                                     |
|  --> Ce que le lecteur doit retenir                       |
|  --> La transformation: avant/apres                      |
|                                                           |
|  CTA (call to action)                                     |
|  --> UN seul CTA par email                               |
|  --> Clair, specifique, actionnable                      |
|  --> "Reponds a cet email" ou "Clique ici pour [X]"     |
|                                                           |
|  P.S.                                                     |
|  --> L'element le PLUS LU apres le subject              |
|  --> Repeter le CTA ou ajouter un bonus                  |
|  --> "P.S. -- Si tu reponds, je te donne aussi [Y]"     |
|                                                           |
+-----------------------------------------------------------+
```

### Le P.S. : L'Arme Secrete

```
POURQUOI LE P.S. EST SI PUISSANT:
+------------------------------------------------------+
|                                                       |
|  Comportement de lecture d'un email:                  |
|                                                       |
|  1. Subject line --> Ouvrir ou pas?                  |
|  2. Preview text --> Confirmer l'interet             |
|  3. Premiere ligne --> Continuer ou fermer?          |
|  4. SCROLL VERS LE BAS --> P.S.                      |
|  5. REMONTE pour lire le corps                        |
|                                                       |
|  Le P.S. est lu par 79% des gens qui ouvrent.       |
|  C'est le 2eme element le plus lu de l'email.        |
|                                                       |
|  UTILISATIONS DU P.S.:                               |
|  - Repeter le CTA principal                          |
|  - Ajouter une urgence                               |
|  - Offrir un bonus surprise                          |
|  - Ajouter un element personnel                      |
|  - Creer de la curiosite pour le prochain email      |
|                                                       |
+------------------------------------------------------+
```

## Segmentation et Personnalisation

### Pourquoi Segmenter

```
SANS SEGMENTATION:
  1 message --> 10,000 personnes --> 2% conversion = 200 ventes

AVEC SEGMENTATION:
  Message A --> 4,000 devs --> 5% conversion = 200 ventes
  Message B --> 3,000 founders --> 4% conversion = 120 ventes
  Message C --> 3,000 freelances --> 3% conversion = 90 ventes
  TOTAL = 410 ventes (+105%)
```

### Segments Essentiels

| Segment | Base Sur | Exemple |
|---------|----------|---------|
| Par interet | Ce qu'ils ont clique/telecharge | "Dev AI" vs "Solopreneur" |
| Par engagement | Ouvertures recentes | Actif (30j) vs Froid (90j+) |
| Par source | D'ou ils viennent | TikTok vs YouTube vs Blog |
| Par achat | Client ou pas | Acheteur vs Prospect |
| Par stade | Debutant vs avance | Junior dev vs Senior |

### Tags et Automations

```
SYSTEME DE TAGS:
+------------------------------------------------------+
|                                                       |
|  Tags automatiques (base sur actions):               |
|  - source:tiktok / source:youtube / source:blog      |
|  - interest:ai / interest:saas / interest:freelance   |
|  - engagement:active / engagement:cold                |
|  - buyer:yes / buyer:no                              |
|  - lead-magnet:checklist / lead-magnet:template       |
|                                                       |
|  Tags manuels (base sur reponses):                   |
|  - segment:developer / segment:founder                |
|  - goal:revenue / goal:productivity                   |
|  - level:beginner / level:advanced                    |
|                                                       |
|  AUTOMATION:                                          |
|  Si clique "AI tools" --> tag interest:ai            |
|  Si ouvre 5+ emails --> tag engagement:active         |
|  Si 0 ouverture 90j --> tag engagement:cold           |
|  Si achete --> tag buyer:yes + sequence onboarding   |
|                                                       |
+------------------------------------------------------+
```

### Les Metriques Cles

| Metrique | Benchmark 2026 | Bon | Excellent |

### A/B Testing Systematique

| Element a Tester | Methode | Impact Potentiel |
|-----------------|---------|-----------------|
| Subject line | Envoyer 2 versions a 20% de la liste, meilleur a 80% | +10-30% opens |
| Heure d'envoi | Tester mardi 9h vs jeudi 11h | +5-15% opens |
| Longueur email | Court (150 mots) vs Long (500 mots) | +10-20% CTR |
| CTA (bouton vs lien) | Bouton HTML vs lien texte | +5-15% CTR |
| Personnalisation | Avec prenom vs sans | +10-20% opens |
| From name | "Gareth" vs "Gareth de Agentik" | +5-10% opens |
| Preview text | Optimise vs par defaut | +5-15% opens |

## Erreurs a Eviter

### Les 10 Erreurs Fatales en Email

| # | Erreur | Consequence | Solution |
|---|--------|-------------|----------|
| 1 | Acheter une liste email | Deliverabilite tuee, blacklist | Construire organiquement |
| 2 | Pas de welcome sequence | 50% des abonnes partent en 48h | Configurer avant de collecter |
| 3 | Envoyer uniquement quand tu vends | Abonnes deconnectes | Emails valeur reguliers |
| 4 | Pas de segmentation | Messages generiques, bas CTR | Segmenter des le debut |
| 5 | Sujet trop long | Coupe sur mobile | Max 50 caracteres |
| 6 | Pas de P.S. | Perte du 2eme element le plus lu | Toujours un P.S. |
| 7 | Multiple CTAs | Confusion, paralysie | Un seul CTA par email |
| 8 | Pas de A/B test | Aucune amelioration | Tester chaque envoi |
| 9 | Ignorer les inactifs | Deliverabilite en baisse | Re-engagement puis nettoyage |
| 10 | Pas de mobile-first | 50%+ des ouvertures | Tester sur telephone |
