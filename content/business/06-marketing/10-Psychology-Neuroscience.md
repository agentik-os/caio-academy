# 1. Les 3 Cerveaux et Comment Ils Décident

## Le Modèle Triunique (MacLean)

```

                                                                 
                      NÉOCORTEX                                
                    (Cerveau Rationnel)                          
                                           
                  • Logique                                    
                  • Analyse                                    
                  • Langage                                    
                  • Planification                              
                                           
                                                                
                                      
                  LIMBIQUE                                   
                (Cerveau Émotionnel)                           
               • Émotions                                      
               • Mémoire                                       
               • Attachement                                   
               • Valeurs                                       
                                      
                                                                
                            
               REPTILIEN                                     
            (Cerveau de Survie)                                
          • Fight / Flight / Freeze                            
          • Instincts de base                                  
          • Sécurité                                           
          • Reproduction                                       
                            
                                                                 
   QUI DÉCIDE VRAIMENT?                                          
     
   Néocortex:  5% des décisions (justifie après coup)            
   Limbique:   50% des décisions (ce qu'on ressent)              
   Reptilien:  45% des décisions (survie, instinct)              
                                                                 

```

## Comment Parler à Chaque Cerveau

### 1. Parler au Reptilien (Survie)

```
CE QUI L'ACTIVE:
• Menaces perçues
• Opportunités de gain rapide
• Signaux de statut social
• Indices de rareté

MOTS ET CONCEPTS:
• "Protège" / "Sécurise" / "Garanti"
• "Maintenant" / "Immédiat"
• "Limité" / "Dernier"
• "Attention" / "Danger" / "Évite"

EXEMPLES:

 "Notre solution optimise vos processus"
 "Protège ton business avant que tes concurrents te dépassent"

 "Inscrivez-vous à notre newsletter"
 "Ne rate pas l'info qui change la donne demain"
```

### 2. Parler au Limbique (Émotions)

```
CE QUI L'ACTIVE:
• Histoires et récits
• Connexion humaine
• Appartenance à un groupe
• Reconnaissance et validation

MOTS ET CONCEPTS:
• "Ressens" / "Imagine" / "Souviens-toi"
• "Ensemble" / "Communauté" / "Famille"
• "Rêve" / "Passion" / "Amour"
• "Fier" / "Accompli" / "Libre"

EXEMPLES:

 "Fonctionnalités premium incluses"
 "Imagine le regard de fierté quand tu lanceras enfin"

 "Rejoignez nos utilisateurs"
 "Rejoins la communauté de founders qui se soutiennent"
```

### 3. Parler au Néocortex (Raison)

```
CE QUI L'ACTIVE:
• Données et statistiques
• Comparaisons logiques
• Preuves et études
• ROI et chiffres

MOTS ET CONCEPTS:
• "Prouvé" / "Étude" / "Recherche"
• "X%" / "Y fois plus" / "Z€ économisés"
• "Logique" / "Stratégie" / "Analyse"
• "Parce que" / "Voici pourquoi"

EXEMPLES:

 "C'est le meilleur du marché"
 "87% de nos clients atteignent leurs objectifs en 90 jours"

 "Très rentable"
 "ROI moyen de 340% sur 12 mois (basé sur 89 projets)"
```

```

                                                                 
   1. REPTILIEN D'ABORD                                          
      Capte l'attention avec urgence/menace

                     ↓

                     ↓

      Justifie avec de la logique

                     ↓                                           
   4. RETOUR AU REPTILIEN                                        
      CTA avec urgence                                           
      "Réserve ton appel maintenant (3 places ce mois)"          
                                                                 

```

# 2. Le Circuit de la Dopamine

## Comment la Dopamine Fonctionne

```

                                                                 
   MYTHE: La dopamine = plaisir de la récompense                 
   RÉALITÉ: La dopamine = anticipation de la récompense          
                                                                 
     
                                                                 
          Dopamine                                               
             ↑                                                   
                                                              
                                                              
                                                              
                                                              
                      ____                                    
                                                              
             → Temps                          
                 ↑       ↑                                       
            Anticipation Récompense                              
                                                                 
   Le pic de dopamine arrive AVANT la récompense,                
   pas pendant. C'est l'ATTENTE qui crée le désir.               
                                                                 

```

## Créer des Pics de Dopamine

### 1. L'Anticipation (Teasing)

```
TECHNIQUE: Révèle progressivement

EXEMPLES:

• "Coming soon..." avec screenshots flous
• "Demain, je révèle quelque chose de gros"
• Countdown avant lancement
• Waitlist avec position visible

POURQUOI ÇA MARCHE:
Le cerveau libère plus de dopamine quand il anticipe
que quand il reçoit. Le teasing prolonge l'anticipation.
```

### 2. Les Récompenses Variables

```
TECHNIQUE: Récompenses imprévisibles

C'EST POURQUOI:

• Les machines à sous sont addictives
• On refresh Twitter compulsivement
• Les loot boxes dans les jeux marchent

APPLICATION MARKETING:

• "Surprise bonus dans cet email" (pas toujours)
• Contenus exclusifs "random" pour subscribers
• Prix variables ("Jusqu'à 50% off" > "30% off fixe")
• Easter eggs dans ton produit
```

### 3. Le Progress et les Milestones

```
TECHNIQUE: Gamification du progrès

EXEMPLES:

• Barre de progression visible
• Badges et achievements
• "Niveau 3 atteint!"
• "Plus que 2 étapes pour..."

APPLICATION:

• Onboarding avec étapes (1/5, 2/5...)
• Referral program avec paliers
• "Tu as lu 47% de ce guide"
• Streaks (jours consécutifs d'utilisation)
```

### 4. La Nouveauté

```
TECHNIQUE: Stimule avec du nouveau

POURQUOI ÇA MARCHE:

Le cerveau libère de la dopamine face à la nouveauté.
C'est un mécanisme de survie (nouvelle info = potentiellement important).

APPLICATION:

• "NOUVEAU:" en subject line
• Feature releases régulières
• "On vient de découvrir..."
• Limited editions
```

## Le Loop Addictif (Hook Model - Nir Eyal)

```

                                                                 
                                                 
                 TRIGGER     ← Externe (notif) ou Interne      
                 (ennui, FOMO)                   
                                                                
                                                                
                                                 
                  ACTION     ← Le comportement le plus simple  
                 (ouvrir app, scroller)          
                                                                
                                                                
                                                 
                RÉCOMPENSE   ← Variable! Pas toujours la même  
                 VARIABLE      (like, contenu intéressant)     
                                                 
                                                                
                                                                
                                                 
              INVESTISSEMENT ← Créer du switching cost         
                 (data, followers, contenu)      
                                                                
                                             
                                                                
                                                
                 TRIGGER     ←                        
                (NOUVEAU)                                      
                                                 
                                                                 

```

### Application pour Tes Apps

```
LIFEPIXELS EXAMPLE:


TRIGGER:
• Externe: Notification "Comment s'est passée ta journée?"
• Interne: Fin de journée = habitude de logging

ACTION:
• Ouvrir l'app (1 tap)
• Logger mood (1 tap)
• Super simple, zéro friction

RÉCOMPENSE VARIABLE:
• Insight surprise sur tes patterns
• Badge débloqué (pas toujours)
• Streak maintenu (+dopamine)
• Comparaison semaine précédente (nouveau!)

INVESTISSEMENT:
• Tes données s'accumulent
• Ton historique devient précieux
• Plus tu utilises, plus les insights sont pertinents
• Tu ne veux pas perdre ton streak

→ Loop qui se renforce avec le temps
```

# 3. Les 25 Biais Cognitifs à Maîtriser

## Catégorie 1: Biais de Décision

### 1. Aversion à la Perte (Loss Aversion)

```
DÉFINITION:
Perdre 100€ fait 2x plus mal que gagner 100€ fait plaisir.

APPLICATION:

 "Gagne 3h par jour avec notre outil"
 "Arrête de perdre 3h par jour sur des tâches manuelles"

 "Économise 500€/mois"
 "Tu perds 500€/mois sans le savoir. Voici comment."

EXEMPLES CONCRETS:
• "Ne perds pas ta place dans la file"
• "Dernière chance avant que le prix augmente"
• "Les features que tu perds en downgrade"
```

### 2. Biais d'Ancrage (Anchoring)

```
DÉFINITION:
Le premier nombre qu'on voit influence tous les jugements suivants.

APPLICATION:

Prix seul: "15,000€" → "C'est cher"

Prix ancré: "Les agences facturent 100K€.
            Moi, c'est 15,000€."
            → "C'est une affaire!"

EXEMPLES CONCRETS:
• Prix barré → nouveau prix
• "Valeur: 5,000€ | Aujourd'hui: 997€"
• Commencer par le package le plus cher
```

### 3. Effet de Dotation (Endowment Effect)

```
DÉFINITION:
On valorise plus ce qu'on possède déjà.

APPLICATION:

• Free trials (ils "possèdent" déjà le produit)
• "Ton dashboard" (pas "le dashboard")
• Customisation (ils investissent dedans)
• "Ne perds pas ton compte configuré"

POURQUOI ÇA MARCHE:
Une fois qu'ils ont configuré, uploadé leurs données, personnalisé...
C'est devenu LEUR produit. Partir = perte.
```

### 4. Biais de Statu Quo

```
DÉFINITION:
On préfère ne rien changer (même si c'est sous-optimal).

APPLICATION:

Pour UTILISER le biais (rétention):
• Rendre le switch difficile (moat)
• Auto-renewal par défaut
• "Continuer avec mon abonnement actuel"

Pour COMBATTRE le biais (acquisition):
• Montrer le coût de l'inaction
• "Chaque mois que tu attends te coûte X€"
• Faciliter la migration (import de data)
```

### 5. Effet de Cadrage (Framing Effect)

```
DÉFINITION:
La façon dont l'info est présentée change la perception.

APPLICATION:

MÊME INFORMATION, FRAMING DIFFÉRENT:

Négatif: "10% de nos clients échouent"
Positif: "90% de nos clients réussissent" ← UTILISE CELUI-LÀ

Perte: "Tu vas perdre 2h/jour"
Gain: "Tu vas gagner 2h/jour" ← DÉPEND DU CONTEXTE

Petit: "Seulement 27€/mois"
Grand: "324€/an" ← UTILISE LE PETIT
```

## Catégorie 2: Biais Sociaux

### 6. Preuve Sociale (Social Proof)

```
DÉFINITION:
Si les autres le font, ça doit être bien.

HIÉRARCHIE DE LA PREUVE SOCIALE:

1. Célébrités/Influenceurs connus    100%
2. Experts reconnus                   80%
3. Amis/Connexions personnelles       70%
4. Utilisateurs similaires            60%
5. Grande quantité ("10K users")      50%
6. Témoignages anonymes               20%

APPLICATION:

• Logos de clients
• Compteur en temps réel ("X personnes regardent")
• Témoignages avec photo, nom, titre
• "Le plus populaire" badge
• Notifications "Pierre vient de s'inscrire"
```

### 7. Autorité

```
DÉFINITION:
On fait confiance aux experts.

APPLICATION:

INDICES D'AUTORITÉ:
• Titres ("CEO de", "Expert en", "10 ans de")
• Credentials (certifications, études)
• Apparitions (médias, podcasts)
• Associations (logos de partenaires)
• Apparence (qualité du site, photos pro)

EXEMPLES:
• "Recommandé par [Expert connu]"
• "Comme vu sur TechCrunch, Forbes..."
• Badges de certifications
• "Développé par d'anciens Google/Meta"
```

### 8. Effet Bandwagon

```
DÉFINITION:
On veut faire partie du groupe dominant.

APPLICATION:

• "Rejoins 10,000+ founders"
• "Le mouvement grandit"
• "Tout le monde en parle"
• Trending badges
• "Ne sois pas le dernier à..."
```

### 9. FOMO (Fear of Missing Out)

```
DÉFINITION:
Peur de rater quelque chose que les autres ont.

APPLICATION:

• "Pendant que tu hésites, 3 concurrents ont lancé"
• Stories de succès d'autres users
• "Les early adopters ont déjà..."
• Notifications de ce que font les autres
• "Tu as raté 3 insights cette semaine"
```

### 10. Liking (Sympathie)

```
DÉFINITION:
On dit oui aux gens qu'on aime.

CE QUI CRÉE LA SYMPATHIE:

• Similarité ("Il est comme moi")
• Compliments sincères
• Contact répété (familiarité)
• Association positive
• Attractivité physique
• Vulnérabilité/Authenticité

APPLICATION:

• Partage ton parcours (points communs)
• Montre tes échecs (vulnérabilité)
• Utilise l'humour
• Photo de profil souriante
• Réponds personnellement
```

## Catégorie 3: Biais de Perception

### 11. Effet de Halo

```
DÉFINITION:
Une qualité positive influence la perception de tout le reste.

APPLICATION:

• Design impeccable → "Le produit doit être bon"
• Un client célèbre → "Ils travaillent avec les meilleurs"
• Une feature impressionnante → toutes semblent bonnes

INVESTIS DANS:
• Design/UX de qualité
• Un témoignage stellar
• Une feature "wow" en démo
```

### 12. Biais de Disponibilité

```
DÉFINITION:
Ce qui vient facilement à l'esprit semble plus fréquent/important.

APPLICATION:

• Être présent régulièrement (top of mind)
• Histoires mémorables (pas stats oubliables)
• Répétition du message
• Trigger words associés à toi

"Quand tu penses MVP → tu penses Gareth"
```

### 13. Effet de Récence

```
DÉFINITION:
Ce qu'on a vu en dernier a plus de poids.

APPLICATION:

• Le plus important en dernier
• Follow-up après une conversation
• Récap des points clés à la fin
• Retargeting (rester présent)
```

### 14. Biais de Confirmation

```
DÉFINITION:
On cherche des infos qui confirment ce qu'on croit déjà.

APPLICATION:

• Identifie ce qu'ils croient déjà
• Confirme-le d'abord
• Puis ajoute ton message

"Tu as raison de vouloir lancer vite.
 Et voici comment je peux t'aider à aller encore plus vite."
```

### 15. Effet Dunning-Kruger

```
DÉFINITION:
Les débutants surestiment leurs capacités.
Les experts sous-estiment les leurs.

APPLICATION:

Pour les débutants:
• Montre-leur ce qu'ils ne savent pas
• "Les 5 erreurs que tu fais sans le savoir"

Pour les experts:
• Valide leur expertise
• Propose de l'amplifier (pas de la remplacer)
```

## Catégorie 4: Biais de Simplification

### 16. Paradoxe du Choix

```
DÉFINITION:
Trop d'options = paralysie et insatisfaction.

APPLICATION:

 7 plans de pricing
 3 plans (un recommandé)

 20 features sur la home
 3 bénéfices principaux

RÈGLE: Maximum 3-4 options
```

### 17. Effet de Défaut

```
DÉFINITION:
Les gens gardent l'option par défaut.

APPLICATION:

• Pré-sélectionne l'option que tu veux
• Annual billing par défaut (plus de revenus)
• Newsletter opt-in par défaut
• Features activées par défaut
```

### 18. Biais de Représentativité

```
DÉFINITION:
On juge par les apparences et les stéréotypes.

APPLICATION:

• Ressemble à ce que tu vends (design pro = produit pro)
• Utilise les codes de ton industrie
• Si tu vends du luxe, présente-toi luxe
• Si tu vends du fun, sois fun
```

### 19. Effet de Contraste

```
DÉFINITION:
On juge par comparaison, pas dans l'absolu.

APPLICATION:

• Montre toujours des alternatives moins bonnes
• "Sans moi" vs "Avec moi" comparaison
• Option "premium" pour que "standard" semble raisonnable
• Avant/Après
```

### 20. Fluency Effect

```
DÉFINITION:
Ce qui est facile à comprendre semble plus vrai et meilleur.

APPLICATION:

• Langage simple (pas de jargon)
• Design clean et lisible
• Messages répétés (familiarité = fluency)
• Police de caractère lisible
```

## Catégorie 5: Biais Temporels

### 21. Hyperbolic Discounting

```
DÉFINITION:
On préfère une récompense petite maintenant à une grande plus tard.

APPLICATION:

• Résultat immédiat ("Commence aujourd'hui, premiers résultats demain")
• Quick wins early dans l'onboarding
• "En moins de 5 minutes, tu verras..."
• Éviter les promesses à long terme seules
```

### 22. Peak-End Rule

```
DÉFINITION:
On se souvient du pic et de la fin, pas du milieu.

APPLICATION:

• Crée un moment "wow" (peak)
• Soigne la dernière impression (end)
• Onboarding: premier succès rapide (peak) + message final positif (end)
• Support: résolution + suivi chaleureux
```

### 23. Planning Fallacy

```
DÉFINITION:
On sous-estime toujours le temps nécessaire.

APPLICATION:

• Sois réaliste sur les délais (crédibilité)
• Livrer plus tôt = surprise positive
• "On prévoit 4 semaines, souvent 3 suffisent"
```

## Catégorie 6: Biais d'Action

### 24. Sunk Cost Fallacy

```
DÉFINITION:
On continue quelque chose parce qu'on a déjà investi.

APPLICATION (RÉTENTION):

• Rappelle ce qu'ils ont déjà accompli
• "Tu as 47 jours de streak"
• "Tu as créé 23 projets"
• "Ne perds pas ton investissement"
```

### 25. Commitment & Consistency

```
DÉFINITION:
Une fois engagé, on veut rester cohérent.

APPLICATION (ESCALADE):

Petits engagements → Grands engagements

1. Follow Twitter (micro)
2. Newsletter signup (petit)
3. Télécharge lead magnet (moyen)
4. Webinar/Demo (engagement temps)
5. Free trial (engagement action)
6. Petit achat (engagement €€)
7. Achat principal (engagement €€€€)
```

# 4. Les 7 Péchés Capitaux du Marketing

## Comment les Utiliser (Éthiquement)

```

                                                                 
   Les 7 péchés sont des motivations humaines fondamentales.     
   Les ignorer = perdre des ventes.                              
   Les exploiter = faire du mal.                                 
   Les comprendre et utiliser éthiquement = marketing efficace.  
                                                                 

```

### 1. ORGUEIL (Pride/Vanité)

```
MOTIVATION: Être admiré, reconnu, supérieur

APPLICATION ÉTHIQUE:

• "Fais partie de l'élite des founders qui..."
• Badges et statuts exclusifs
• "Impressionne tes investisseurs"
• Case studies où le client brille
• "Montre au monde ce que tu sais faire"

EXEMPLES:
• "Premium member" status
• Certifications à afficher
• Leaderboards
• Features de partage de succès
```

### 2. AVARICE (Greed/Cupidité)

```
MOTIVATION: Avoir plus (argent, possessions, pouvoir)

APPLICATION ÉTHIQUE:

• Promesses de gain réalistes
• ROI et multiplicateurs
• "Débloque de nouvelles sources de revenus"
• Économies démontrables
• Plus de features, plus de valeur

EXEMPLES:
• "Multiplie tes revenus par X"
• "Économise X€/mois"
• "Lifetime deal" (posséder pour toujours)
• Bundles avec plus de valeur
```

### 3. LUXURE (Lust)

```
MOTIVATION: Plaisir, désir, attractivité

APPLICATION ÉTHIQUE:

• Design beau et désirable
• Expérience agréable à utiliser
• "Tu vas adorer utiliser ça"
• Satisfaction sensorielle (UI smooth)
• Screenshots sexy du produit

EXEMPLES:
• UI/UX premium
• Micro-interactions plaisantes
• Animations fluides
• "Designed with love"
```

### 4. ENVIE (Envy)

```
MOTIVATION: Vouloir ce que les autres ont

APPLICATION ÉTHIQUE:

• Témoignages de succès
• "Comment [personne similaire] a réussi"
• Features "utilisées par [entreprise cool]"
• Ce que font les concurrents avec ça

EXEMPLES:
• "Comme utilisé par Stripe, Notion..."
• Stories de transformation
• "Tes concurrents utilisent déjà..."
• Comparaisons avant/après
```

### 5. GOURMANDISE (Gluttony)

```
MOTIVATION: Consommer plus, excès

APPLICATION ÉTHIQUE:

• All-you-can-eat pricing
• "Illimité"
• Bundles massifs
• Bonus sur bonus
• "Tout inclus"

EXEMPLES:
• "Unlimited users/projects"
• "Accès à tout le catalogue"
• "10 bonus inclus"
• Lifetime access
```

### 6. PARESSE (Sloth)

```
MOTIVATION: Facilité, moins d'effort

APPLICATION ÉTHIQUE:

• "Sans effort"
• "Automatique"
• "On s'occupe de tout"
• "En 3 clics"
• Templates pré-faits

EXEMPLES:
• "Set it and forget it"
• Auto-pilot features
• "Done for you"
• 1-click solutions
• "Zero maintenance"
```

### 7. COLÈRE (Wrath)

```
MOTIVATION: Contre injustice, frustration

APPLICATION ÉTHIQUE:

• Contre un ennemi commun
• "Assez de [frustration]!"
• "Tu mérites mieux que ça"
• Validation de leur frustration
• "Reprends le contrôle"

EXEMPLES:
• "Marre des agences qui surfacturent?"
• "Stop aux outils compliqués"
• "Révolte contre la médiocrité"
• "Tu mérites un vrai partenaire, pas un prestataire"
```

# 5. Neuroscience de l'Attention

## Comment le Cerveau Filtre l'Information

```

                                                                 
   INFORMATIONS REÇUES: 11,000,000 bits/seconde                  
   TRAITÉES CONSCIEMMENT: 40 bits/seconde                        
                                                                 
   = Le cerveau filtre 99.9996% de l'information                 
                                                                 
   CE QUI PASSE LE FILTRE:                                       
     
   1. Menaces (survie)                                           
   2. Nouveauté                                                  
   3. Ce qui est pertinent (pour mes goals)                      
   4. Mouvement                                                  
   5. Contraste                                                  
   6. Émotion                                                    
                                                                 

```

## Les 6 Stimuli d'Attention

### 1. Pattern Interrupt (Nouveauté/Contraste)

```
TECHNIQUE: Casser le pattern attendu

EXEMPLES:

• Headline inattendue ("La pire chose que j'ai faite")
• Design différent des concurrents
• Couleur qui contraste
• Format inhabituel
• "Attention: ceci n'est pas pour tout le monde"
```

### 2. Mouvement

```
TECHNIQUE: Le mouvement capte l'attention primitive

EXEMPLES:

• Vidéos auto-play (avec soin)
• Animations subtiles
• Carrousels
• Progress indicators
• Notifications qui pop
```

### 3. Visages et Yeux

```
TECHNIQUE: On regarde instinctivement les visages

EXEMPLES:

• Photos avec regards vers le CTA
• Témoignages avec photos
• Ta photo sur la page
• Expressions émotionnelles
• Pas de photos stock robotiques
```

### 4. Le Nombre 3

```
TECHNIQUE: Le cerveau aime les groupes de 3

EXEMPLES:

• 3 bénéfices principaux
• 3 étapes du process
• 3 plans de pricing
• 3 témoignages
• 3 raisons de...
```

### 5. Open Loops

```
TECHNIQUE: Questions non résolues créent de la tension

EXEMPLES:

• "La technique #3 va te surprendre..."
• "Mais d'abord, laisse-moi te raconter..."
• "Ce que j'ai découvert m'a choqué"
• Cliffhangers en fin de section

POURQUOI ÇA MARCHE:
Le cerveau a BESOIN de fermer les boucles (Zeigarnik Effect).
On ne peut pas s'empêcher de continuer à lire.
```

### 6. Personnalisation

```

• "[Prénom], tu as une nouvelle notification"
• Dashboard personnalisé

• "Pour toi, [Prénom]"
```

# 6. Psychologie de la Confiance

## Les 5 Piliers de la Confiance

```

                                                                 
   CONFIANCE = Compétence × Fiabilité × Intimité / Risque perçu  
                                                                 
     
                                                                 
   COMPÉTENCE (Tu sais faire)                                    
    Credentials                                               
    Track record                                              
    Expertise démontrée                                       
    Qualité du travail visible                                
                                                                 
   FIABILITÉ (Tu fais ce que tu dis)                             
    Tenir ses promesses                                       
    Délais respectés                                          
    Cohérence dans le temps                                   
    Garanties                                                 
                                                                 
   INTIMITÉ (Je me sens à l'aise)                                
    Connexion humaine                                         
    Vulnérabilité partagée                                    
    Communication empathique                                  
    Confidentialité                                           
                                                                 
   RISQUE PERÇU (Ce que je risque de perdre)                     
    Argent                                                    
    Temps                                                     
    Réputation                                                
    Opportunité coût                                          
                                                                 

```

## Comment Construire la Confiance Rapidement

### 1. Preuve de Compétence

```
TECHNIQUES:

• Case studies détaillées (process + résultats)
• Portfolio visible
• Témoignages spécifiques (pas vagues)
• Démontre en live (webinars, démos)
• Contenu éducatif (prouve que tu maîtrises)
• "89 apps lancées. 73 rentables."
```

### 2. Preuve de Fiabilité

```
TECHNIQUES:

• Garanties claires
• Délais explicites (et tenus)
• Réponse rapide aux messages
• Cohérence du message dans le temps
• Transparence sur les limitations
• Reviews et ratings publics
```

### 3. Création d'Intimité

```
TECHNIQUES:

• Partage tes échecs (vulnérabilité)
• Story personnelle
• Utilise le prénom
• Empathie ("Je comprends, j'ai vécu ça")
• Réponds personnellement (pas des bots)
• Ton conversationnel (pas corporate)
```

### 4. Réduction du Risque Perçu

```
TECHNIQUES:

• Garantie satisfait ou remboursé
• Free trial (zéro engagement)
• "Annule quand tu veux"
• Pas de CB requise au début
• FAQ qui adresse les peurs
• Prix fixe (pas de surprise)
• Témoignages de sceptiques convertis
```

# 7. Science de la Persuasion (Cialdini Expanded)

## Les 6+1 Principes de Cialdini

### 1. Réciprocité (Expanded)

```
PRINCIPE:
Quand on reçoit quelque chose, on se sent obligé de rendre.

3 TYPES DE RÉCIPROCITÉ:

1. MATÉRIELLE: Tu donnes quelque chose de tangible
   → Lead magnets, free tools, templates

2. INFORMATIONNELLE: Tu donnes de la connaissance
   → Conseils gratuits, contenu éducatif, insights

3. ÉMOTIONNELLE: Tu donnes du temps, de l'attention
   → Réponses personnalisées, écoute, feedback

NIVEAU DE DETTE (du plus faible au plus fort):
• Contenu public → faible
• Email personnalisé → moyen
• Cadeau inattendu → fort
• Aide dans une situation difficile → très fort
```

### 2. Engagement & Cohérence (Expanded)

```
PRINCIPE:
Une fois engagé publiquement, on veut rester cohérent.

TECHNIQUES D'ESCALADE:

1. PIED DANS LA PORTE
   Petit oui → grand oui
   "Tu peux répondre à 1 question?" → "Tu peux faire un call?"

2. ENGAGEMENT PUBLIC
   Ils annoncent → ils tiennent
   "Partage ton objectif avec la communauté"

3. ENGAGEMENT ÉCRIT
   Plus fort que verbal
   "Signe ici que tu t'engages à..."

4. AUTO-IDENTIFICATION
   "Tu es du genre à..."
   → Ils veulent prouver que oui
```

### 3. Preuve Sociale (Expanded)

```
PRINCIPE:
On fait ce que font les autres similaires à nous.

LES 5 TYPES DE PREUVE SOCIALE:

1. EXPERT
   "[Expert reconnu] recommande..."

2. CÉLÉBRITÉ
   "[Personne connue] utilise..."

3. USER
   "[Nombre] personnes utilisent..."

4. SAGESSE DES FOULES
   "Le plus populaire" / "Trending"

5. PAIRS
   "[Personnes similaires à toi] font..."

PERTINENCE > QUANTITÉ:
"10 founders tech utilisent..." > "10,000 personnes utilisent..."
(si ta cible = founders tech)
```

### 4. Autorité (Expanded)

```
PRINCIPE:
On obéit aux figures d'autorité.

LES MARQUEURS D'AUTORITÉ:

TITRES:
• "Dr.", "CEO", "Expert en", "Auteur de"

VÊTEMENTS (MÉTAPHORIQUES):
• Design professionnel du site
• Qualité du contenu
• Présentation soignée

SYMBOLES:
• Certifications
• Mentions presse
• Awards
• Associations prestigieuses

CONTENU:
• Livres publiés
• Conférences données
• Études citées
```

### 5. Sympathie (Expanded)

```
PRINCIPE:
On dit oui aux gens qu'on aime.

LES 5 FACTEURS DE SYMPATHIE:

1. ATTRACTIVITÉ PHYSIQUE
   • Photo pro et souriante
   • Design agréable

2. SIMILARITÉ
   • Points communs soulignés
   • "Je suis passé par là aussi"

3. COMPLIMENTS
   • Sincères, pas flatteurs
   • "J'admire ce que tu construis"

4. CONTACT RÉPÉTÉ
   • Familiarité = sympathie
   • Présence régulière

5. ASSOCIATION POSITIVE
   • Associé à des succès, pas des échecs
   • Partenariats avec des marques aimées
```

### 6. Rareté (Expanded)

```
PRINCIPE:
Ce qui est rare a plus de valeur.

3 TYPES DE RARETÉ:

1. QUANTITÉ LIMITÉE
   "Il ne reste que 5 places"
   "Édition limitée à 100 exemplaires"

2. TEMPS LIMITÉ
   "Offre expire dans 48h"
   "Early bird jusqu'à vendredi"

3. EXCLUSIVITÉ
   "Réservé aux membres"
   "Sur invitation uniquement"

AUTHENTICITÉ CRUCIALE:
Fausse rareté = destruction de confiance
Vraie rareté = puissant
```

### 7. Unité (Le 7ème principe)

```
PRINCIPE:
On dit oui à ceux de notre "tribu".

APPLICATION:

• "Entre founders, on se comprend"
• Vocabulaire de groupe
• Ennemis communs
• Valeurs partagées
• "Nous" vs "Eux"

EXEMPLES:
• "La communauté des indie hackers"
• "Nous, les builders qui n'ont pas levé de fonds"
• "Contrairement aux agences traditionnelles..."
```

# 8. Triggers Émotionnels Avancés

## La Carte des Émotions Marketing

```

                                                                 
   ÉMOTIONS HIGH-AROUSAL (ACTION)    vs    LOW-AROUSAL (PASSIF)  
                 
    Colère                               Tristesse            
    Peur/Anxiété                         Contentement         
    Excitation                           Ennui                
    Surprise/Awe                         Satisfaction calme   
    Frustration                          Mélancolie           
                                                                 
   UTILISE HIGH-AROUSAL POUR:                                    
   • Partages viraux                                             
   • Passages à l'action immédiats                               
   • Headlines et hooks                                          
                                                                 
   UTILISE LOW-AROUSAL POUR:                                     
   • Relation long-terme                                         
   • Fidélisation                                                
   • Post-achat (satisfaction)                                   
                                                                 

```

## Les 12 Triggers Émotionnels

### 1. Frustration → Soulagement

```
SÉQUENCE:
1. Rappelle la frustration ("Tu en as marre de...")
2. Amplifie ("Ça te coûte X€/mois")
3. Propose le soulagement ("Enfin une solution qui...")

EXEMPLE:
"Tu en as marre de passer des heures sur des tâches répétitives?
Chaque jour, tu perds 3h de travail qui pourrait être automatisé.
Imagine récupérer ce temps pour ce qui compte vraiment."
```

### 2. Peur → Sécurité

```
SÉQUENCE:
1. Identifie la peur ("Et si ton projet échouait...")
2. Normalise ("80% des startups échouent")
3. Propose la sécurité ("Avec nous, réduction du risque")

ATTENTION:
Utilise la peur avec parcimonie. Trop = manipulation.
```

### 3. Doute → Certitude

```
SÉQUENCE:
1. Valide le doute ("Tu te demandes si c'est le bon moment")
2. Montre les preuves ("Voici les résultats de 89 projets")
3. Crée la certitude ("Garanti ou remboursé")
```

### 4. Confusion → Clarté

```
SÉQUENCE:
1. Reconnaître la confusion ("Le monde du dev est complexe")
2. Simplifier ("En réalité, 3 choses comptent")
3. Guider ("Voici le chemin clair")
```

### 5. Solitude → Appartenance

```
SÉQUENCE:
1. Comprendre l'isolement ("Construire seul, c'est dur")
2. Montrer la communauté ("10,000 comme toi")
3. Inviter ("Rejoins-nous")
```

### 6. Ennui → Excitation

```
SÉQUENCE:
1. Reconnaître la routine ("Ton job est devenu prévisible")
2. Montrer l'aventure ("Imagine lancer ton propre truc")
3. Créer l'excitement ("L'aventure commence maintenant")
```

### 7. Impatience → Satisfaction

```
SÉQUENCE:
1. Reconnaître l'attente ("Tu veux des résultats, pas dans 6 mois")
2. Promettre la vitesse ("En 4 semaines")
3. Livrer ("Premiers résultats dès la semaine 1")
```

### 8. Culpabilité → Rédemption

```
SÉQUENCE:
1. Reconnaître ("Tu repousses ce projet depuis longtemps")
2. Déculpabiliser ("La vie, le travail... c'est normal")
3. Offrir la rédemption ("Aujourd'hui, tu changes ça")
```

### 9. Envie → Émulation

```
SÉQUENCE:
1. Montrer le succès des autres ("le CTO SaaS a lancé en 3 semaines")
2. Créer le pont ("Il était comme toi il y a 6 mois")
3. Inviter à l'action ("Ton tour maintenant")
```

### 10. Honte → Fierté

```
SÉQUENCE:
1. Comprendre la honte ("Tu parles de ton projet depuis 2 ans")
2. Normaliser ("L'exécution est la partie dure")
3. Promettre la fierté ("Imagine annoncer ton lancement")
```

### 11. Scepticisme → Conviction

```
SÉQUENCE:
1. Valider le scepticisme ("Tu as raison de douter")
2. Preuves irréfutables ("Voici les chiffres, vérifiables")
3. Convertir ("Convaincu? Voici le premier pas")
```

### 12. Indifférence → Curiosité

```
SÉQUENCE:
1. Pattern interrupt ("Ce que je vais te dire va te choquer")
2. Open loop ("La raison pour laquelle 90% échouent...")
3. Promise ("Tu vas découvrir le secret que...")
```

# 9. Psychologie du Prix

## La Perception du Prix

```

                                                                 
   LE PRIX N'EST PAS UN CHIFFRE.                                 
   C'EST UNE PERCEPTION.                                         
                                                                 
   Le même service:                                              
   • 10,000€ seul → "C'est cher"                                 
   • 10,000€ vs 100,000€ d'agence → "C'est pas cher"             
   • 10,000€ pour 100,000€ de résultats → "C'est gratuit"        
                                                                 

```

## Techniques de Pricing Psychologique

### 1. L'Ancrage

```
TECHNIQUE:
Montre d'abord un prix plus élevé

EXEMPLES:

• "Valeur: 5,000€ | Aujourd'hui: 997€"
• "Agence: 100K€ | Moi: 15K€"
• Montre le plan "Enterprise" en premier
• Prix barré visible
```

### 2. Le Charm Pricing (.99)

```
TECHNIQUE:
Finir par 9 ou 7

EXEMPLES:

• 99€ au lieu de 100€
• 997€ au lieu de 1000€
• 47€ au lieu de 50€

POURQUOI ÇA MARCHE:
Le cerveau lit de gauche à droite.
99€ est perçu comme "90 quelque chose", pas "presque 100".
```

### 3. Le Price Decoy

```
TECHNIQUE:
Une option existe juste pour rendre l'autre attractive

EXEMPLE:

Basic: 9€/mois     (fonctionnalités limitées)
Pro: 29€/mois      (tout inclus) ← ON VEUT VENDRE ÇA
Enterprise: 99€/mois (features que personne n'utilise)

L'option Enterprise rend Pro raisonnable.
```

### 4. Le Bundle

```
TECHNIQUE:
Plusieurs choses ensemble à prix unique

POURQUOI ÇA MARCHE:

• Difficile de comparer (pas de référent)
• Valeur perçue augmente
• "Tout ça pour seulement..."

EXEMPLE:
"Le Kit Complet: Guide + Templates + Checklist + Bonus"
Séparé: 47€ + 29€ + 19€ + 27€ = 122€
Bundle: 67€
```

### 5. Le Prix Unitaire

```
TECHNIQUE:
Divise le prix en petites unités

EXEMPLES:

 "1200€/an"
 "100€/mois"
 "3,33€/jour"
 "Moins qu'un café par jour"
```

### 6. Le Freemium

```
TECHNIQUE:
Version gratuite → Version payante

PSYCHOLOGIE:

• Zéro risque pour essayer
• Ils "possèdent" déjà (endowment effect)
• Sunk cost (temps investi)
• Frustration calibrée (la version gratuite DOIT frustrer juste assez)
```

### 7. La Garantie Inversée

```
TECHNIQUE:
La garantie augmente la valeur perçue

EXEMPLES:

• "Satisfait ou remboursé 30 jours"
• "Double ton argent si pas de résultats"
• "Garanti à vie"

PSYCHOLOGIE:
Plus la garantie est forte, plus le prix semble justifié.
"Ils sont sûrs d'eux" → "Ça doit marcher"
```

# 10. Applications Pratiques par Contexte

## Landing Page

```
PSYCHOLOGIE À APPLIQUER:


ABOVE THE FOLD (5 secondes):
• Pattern interrupt (headline)

MILIEU DE PAGE:
• Limbique (story, émotions)
• Biais cognitifs (social proof, autorité)
• Open loops (curiosité pour scroller)

BAS DE PAGE:
• Néocortex (FAQ, détails logiques)
• Rareté/Urgence
• Risk reversal (garantie)
• CTA final (reptilien + urgence)
```

## Email Marketing

```
PSYCHOLOGIE À APPLIQUER:


SUBJECT LINE:
• Curiosité (open loops)
• Personnalisation ([Prénom])
• Urgence si approprié

OPENING:
• Hook émotionnel
• Pattern interrupt
• Personnalisation

BODY:
• Une émotion dominante
• Story/exemples
• Bénéfices (pas features)

CTA:
• Un seul, clair
• Action verb + bénéfice
• Urgence/rareté si approprié

P.S.:
• Second hook
• Rappel du bénéfice ou de l'urgence
```

## Sales Call

```
PSYCHOLOGIE À APPLIQUER:


OPENING (2 min):
• Rapport building (similarité, liking)
• Démontrer intérêt sincère

DÉCOUVERTE (10 min):
• Questions (ils parlent 80%)
• Écoute active
• Identifier les émotions (peur, frustration, désir)

PRÉSENTATION (10 min):
• Leurs mots, pas les tiens
• Bénéfices personnalisés
• Preuves pertinentes pour EUX

OBJECTIONS:
• Feel-Felt-Found
• Ne pas combattre, comprendre
• Risk reversal

CLOSE:
• Assumptive close ("Quand veux-tu commencer?")
• Urgence vraie si applicable
• Silence après la question
```

## Tweet/Thread

```
PSYCHOLOGIE À APPLIQUER:


HOOK (1ère ligne):
• Pattern interrupt
• Curiosité/Open loop
• Promesse de valeur
• Émotion high-arousal

BODY:
• Une idée par tweet
• Bucket brigades ("Mais voici le truc...")
• Stories et exemples
• Spécificité (chiffres, détails)

CTA (dernier tweet):
• Réciprocité ("Si ce thread t'a aidé...")
• Social proof ("Rejoins X followers")
• Simplicité ("Follow pour plus")
```
