### Pourquoi un Dev/Solopreneur DOIT y Etre

```
AVANT (2020-2022):
  Dev --> Blog technique --> SEO --> Clients en 6-12 mois

MAINTENANT (2025-2026):
  Dev --> 1 video TikTok virale --> 100K vues en 24h --> Clients en 1 semaine

Le short-form est le SEUL format ou un inconnu
peut atteindre des millions de personnes SANS audience existante.
```

**Raisons specifiques pour un dev:**
- Les gens ADORENT voir du code a l'ecran (effet "matrix")
- Le build-in-public est un format natif du short-form
- Les AI tools/automations fascinent le grand public
- Le solopreneur lifestyle est aspirationnel
- Tu montres ton expertise SANS vendre (pull marketing)

## L'Algorithme : Comment il Fonctionne

### TikTok - Le Systeme de Distribution

```
+-- ETAPE 1: Upload -------------------------------------------+
|                                                               |
|  Ta video est montree a un PETIT GROUPE (100-300 personnes)  |
|  Ce groupe est compose principalement de tes FOLLOWERS        |
|  (changement majeur 2026 vs 2024)                            |
|                                                               |
+-- ETAPE 2: Evaluation --------------------------------------+
|                                                               |
|  L'algo mesure:                                               |
|  - Watch time / Completion rate (LE PLUS IMPORTANT)          |
|  - Ratio likes/vues                                           |
|  - Commentaires et partages                                   |
|  - Saves (signal tres fort)                                   |
|  - Profil visits apres visionnage                            |
|                                                               |
+-- ETAPE 3: Distribution ------------------------------------+
|                                                               |
|  SI les metriques sont bonnes:                                |
|  Pool 1 --> 1K-5K vues                                       |
|  Pool 2 --> 10K-50K vues                                     |
|  Pool 3 --> 100K-500K vues                                   |
|  Pool 4 --> 1M+ vues (viralite)                              |
|                                                               |
|  A CHAQUE pool, les metriques sont re-evaluees.              |
|  Si ca baisse --> la distribution s'arrete.                   |
|                                                               |
+---------------------------------------------------------------+
```

### Changements Cles 2026

| Signal | 2024 | 2026 |
|--------|------|------|
| Completion rate pour viralite | 50%+ | 70%+ |
| Duree optimale | 15-30s | 60-180s |
| Test initial | Random viewers | Tes followers d'abord |
| Importance followers | Faible | Forte (premiere evaluation) |
| Contenu original | Prefere | Obligatoire (detection IA) |
| Partages | Important | Signal #1 (au dessus des likes) |

## L'Anatomie d'une Video Virale

### Structure en 3 Parties

```
+-- HOOK (0-3 secondes) ------- 50% de l'importance -----+
|                                                          |
|  OBJECTIF: Empecher le scroll                           |
|  REGLE: Si tu perds ici, RIEN d'autre ne compte        |
|                                                          |
+-- RETENTION (3s - fin-5s) --- 35% de l'importance -----+
|                                                          |
|  OBJECTIF: Garder l'attention jusqu'au bout             |
|  REGLE: Chaque seconde doit apporter de la valeur      |
|         ou creer du suspense                             |
|                                                          |
+-- CTA (dernieres 3-5s) ------ 15% de l'importance -----+
|                                                          |
|  OBJECTIF: Provoquer une action                         |
|  REGLE: Un seul CTA, clair et simple                    |
|                                                          |
+----------------------------------------------------------+
```

## Algorithme et Croissance

### Les Signaux de l'Algo par Priorite

```
CLASSEMENT DES SIGNAUX (du plus au moins important):

1. [++++++++++] PARTAGES (signal #1 en 2026)
   --> La video est envoyee en DM ou partagee
   --> Signal de "contenu qui MERITE d'etre diffuse"

2. [+++++++++.] WATCH TIME / COMPLETION RATE
   --> % de la video regardee
   --> Objectif: 70%+ pour viralite

3. [++++++++..] COMMENTAIRES
   --> Surtout les commentaires longs
   --> Les reponses du createur comptent aussi

   --> Le signal le MOINS important
   --> Mais toujours compte dans l'equation

7. [++++......] FOLLOWS apres visionnage
   --> Tu apportes assez de valeur pour suivre
```

## Erreurs a Eviter

### Les 10 Erreurs Fatales

| # | Erreur | Consequence | Solution |
|---|--------|-------------|----------|
| 1 | Poster sans hook | 0% retention | Toujours commencer par le hook |
| 2 | Videos trop longues sans valeur | Drop-off massif | Chaque seconde doit servir |
| 3 | Ne pas poster regulierement | L'algo t'oublie | Minimum 5x/semaine |
| 4 | Copier les trends sans adapter | Pas d'authenticite | Adapter a ta niche |
| 5 | Ignorer les analytics | Pas d'amelioration | Review hebdo obligatoire |
| 6 | Vendre trop tot | Perte de confiance | 80/20 valeur/promo |
| 7 | Pas de sous-titres | -40% retention | Toujours des sous-titres |
| 8 | Cross-post avec watermark | Penalise par l'algo | Retirer les watermarks |
| 9 | Attendre la perfection | 0 video publiee | Fait > Parfait |
| 10 | Ne pas engager en commentaires | Pas de communaute | Repondre a tout |
