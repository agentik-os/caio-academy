# 3. Workflow Automatise

## Saturday Video Workflow

```
+------------------------------------------------------------------+
|                                                                   |
|  SATURDAY WORKFLOW: VIDEO PRODUCTION                             |
|  ====================================                            |
|                                                                   |
|  09:00-10:00  PREP                                               |
|  +----------------------------------------------------------+   |
|  | 1. Ouvre le script (ecrit pendant la semaine)             |   |
|  | 2. Setup recording (camera, mic, lumiere)                 |   |
|  | 3. Teste audio/video                                      |   |
|  | 4. Warm up voix                                           |   |
|  +----------------------------------------------------------+   |
|                                                                   |
|  10:00-12:00  RECORDING                                          |
|  +----------------------------------------------------------+   |
|  | 1. Record main video (1-2 takes max)                      |   |
|  | 2. Record B-roll / screen recordings                      |   |
|  | 3. Record short hooks separes (pour shorts)               |   |
|  +----------------------------------------------------------+   |
|                                                                   |
|  12:00-13:00  BREAK                                              |
|                                                                   |
|  14:00-15:30  AI PROCESSING                                      |
|  +----------------------------------------------------------+   |
|  | 1. Upload video vers Descript                             |   |
|  | 2. Auto-transcription                                     |   |
|  | 3. Remove filler words (auto)                             |   |
|  | 4. Export transcription -> Claude                         |   |
|  | 5. Claude: Transform en thread + tweets + LinkedIn        |   |
|  +----------------------------------------------------------+   |
|                                                                   |
|  15:30-17:00  VIDEO EDITING                                      |
|  +----------------------------------------------------------+   |
|  | 1. Rough cut dans Descript                                |   |
|  | 2. Export vers OpusClip pour shorts                       |   |
|  | 3. Select top 5 shorts                                    |   |
|  | 4. Add captions (auto)                                    |   |
|  +----------------------------------------------------------+   |
|                                                                   |
|  17:00-18:00  GRAPHICS & SCHEDULING                              |
|  +----------------------------------------------------------+   |
|  | 1. Thumbnail dans Canva                                   |   |
|  | 2. Quote graphics (5)                                     |   |
|  | 3. Schedule YouTube video                                 |   |
|  | 4. Schedule shorts (1/jour)                               |   |
|  +----------------------------------------------------------+   |
|                                                                   |
|  OUTPUT:                                                         |
|  |-- 1 long-form video (ready to publish)                       |
|  |-- 5 shorts (scheduled for the week)                          |
|  |-- 5 quote graphics                                           |
|  |-- Transcript for Sunday text content                         |
|  +-- Thread draft for review                                    |
|                                                                   |
+------------------------------------------------------------------+
```

# 4. Prompts Templates

## Prompt: Video to Twitter Thread

```
+------------------------------------------------------------------+
|                                                                   |
|  PROMPT: VIDEO TRANSCRIPT -> TWITTER THREAD                      |
|  ==========================================                      |
|                                                                   |
|  Tu es un expert en content marketing Twitter.                   |
|  Transforme cette transcription video en un thread Twitter       |
|  viral de 10-15 tweets.                                         |
|                                                                   |
|  REGLES:                                                         |
|  - Premier tweet = hook puissant (question ou statement choc)   |
|  - Chaque tweet = 1 idee complete                               |
|  - Utilise des line breaks pour la lisibilite                   |
|  - Termine par un CTA (follow, newsletter, etc)                 |
|  - Pas d'emojis                                                 |
|  - Tone: Direct, personnel, authentique                         |
|                                                                   |
|  STRUCTURE:                                                      |
|  Tweet 1: Hook                                                   |
|  Tweet 2-9: Points cles (1 par tweet)                           |
|  Tweet 10: Recapitulatif                                        |
|  Tweet 11: CTA                                                  |
|                                                                   |
|  TRANSCRIPTION:                                                  |
|  [COLLER LA TRANSCRIPTION]                                       |
|                                                                   |
+------------------------------------------------------------------+
```

```
+------------------------------------------------------------------+
|                                                                   |
|  PROMPT: TWITTER THREAD -> LINKEDIN POST                         |
|  ========================================                        |
|                                                                   |
|  Transforme ce thread Twitter en post LinkedIn.                  |

|  - Story/Context (2-3 paragraphes)                              |

|  - Question finale                                              |

|  THREAD:                                                         |
|  [COLLER LE THREAD]                                              |
|                                                                   |

```
