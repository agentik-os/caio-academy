## 5. Video Structure Template

### Universal Structure (All Categories)

```
+-- VIDEO STRUCTURE ----------------------------------------+
|                                                           |
|  HOOK (0:00 - 0:30)         [==]  ~5% of video          |
|  |-- First 5 seconds: visual/verbal HOOK                 |
|  |-- Show the RESULT or make a BOLD CLAIM                |
|  +-- "By the end of this video, you'll..."              |
|                                                           |
|  INTRO (0:30 - 1:30)        [==]  ~8% of video          |
|  |-- Establish credibility (brief)                       |
|  |-- Context: why this matters NOW                       |
|  +-- Roadmap: "Here's what we'll cover..."              |
|                                                           |
|  CONTENT (1:30 - [end-3min]) [================]  ~75%    |
|  |-- 3-5 content blocks (chapters)                       |
|  |-- Each block: teach -> show -> summarize              |
|  |-- Pattern interrupts every 3-4 minutes               |
|  +-- Key insight per block                              |
|                                                           |
|  BRIDGE (last 3 min)        [==]  ~12% of video         |
|  |-- Summarize 3 key takeaways                           |
|  |-- Tease next video                                    |
|  |-- CTA: subscribe + specific next video                |
|  +-- End screen (card to next video)                    |
|                                                           |
+-----------------------------------------------------------+
```

### Hook Formulas (First 5 Seconds)

| Type | Formula | Example |

| **Result First** | Show the end result | "This is a fully working AI agent that handles customer support. I built it in 45 minutes." |
| **Bold Claim** | Make a specific claim | "198 AI agents. That's how many I run across 6 products. Let me show you how." |
| **Contrarian** | Challenge assumptions | "Everyone's building AI wrappers. That's the wrong approach. Here's why." |
| **Question** | Ask a burning question | "What if I told you a single developer could out-ship a team of 10?" |
| **Story** | Start mid-action | "Last Tuesday at 3am, one of my agents found a critical bug. Here's what happened." |
