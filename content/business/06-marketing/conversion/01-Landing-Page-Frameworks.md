# 2. Les Frameworks de Structure

## Framework PAS (Problem-Agitate-Solution)

```
+------------------------------------------------------------------+
|                                                                   |
|  FRAMEWORK PAS                                                   |
|  =============                                                   |
|                                                                   |
|  P - PROBLEM                                                     |
|      Identifie le probleme clairement                            |
|      "Tu perds des heures chaque semaine a..."                  |
|                                                                   |
|  A - AGITATE                                                     |
|      Appuie sur la douleur, rends-la urgente                    |
|      "Et chaque jour qui passe, tu..."                          |
|                                                                   |
|  S - SOLUTION                                                    |
|      Presente ta solution comme l'evidence                       |
|      "C'est pourquoi nous avons cree..."                        |
|                                                                   |
|  EXEMPLE AGENTIK OS:                                             |
|  ====================                                            |
|                                                                   |
|  P: "Tu as des objectifs ambitieux mais tu n'arrives pas        |
|      a rester constant. Tes apps de tracking sont trop          |
|      compliquees ou trop basiques."                             |
|                                                                   |
|  A: "Chaque mois sans systeme, c'est du potentiel perdu.        |
|      Tu sais ce que tu devrais faire, mais la friction          |
|      te fait abandonner. Encore et encore."                     |
|                                                                   |
|  S: "LifePixels utilise l'AI pour comprendre tes patterns       |
|      et t'aider a progresser sans effort. Pas de setup          |
|      complique. Juste des resultats."                           |
|                                                                   |
+------------------------------------------------------------------+
```

## Framework AIDA (Attention-Interest-Desire-Action)

```
+------------------------------------------------------------------+
|                                                                   |
|  FRAMEWORK AIDA                                                  |
|  ==============                                                  |
|                                                                   |
|  A - ATTENTION                                                   |
|      Headline qui arrete le scroll                               |
|      Quelque chose de surprenant ou provocant                    |
|                                                                   |
|  I - INTEREST                                                    |
|      Developpe avec des details intrigants                       |
|      "Voici ce qui est different..."                            |
|                                                                   |
|  D - DESIRE                                                      |
|      Cree le desir avec benefits + proof                         |
|      Temoignages, resultats, visualization                       |
|                                                                   |
|  A - ACTION                                                      |
|      CTA clair et irresistible                                   |
|      "Start free now"                                           |
|                                                                   |
|  EXEMPLE DAFNCK STUDIO:                                          |
|  ======================                                          |
|                                                                   |
|  A: "Ton MVP en 4 semaines. Pas 4 mois."                        |
|                                                                   |
|  I: "Avec le developpement AI-accelerated, je livre             |
|      des produits production-ready en une fraction              |
|      du temps des agences traditionnelles."                     |
|                                                                   |
|  D: "73 apps lancees, 15M generes pour mes clients.             |
|      Voici ce que dit le Consultant AI, founder de XYZ:                     |
|      'Gareth a livre en 5 semaines ce que notre                 |
|      precedente agence n'a pas fait en 6 mois.'"                |
|                                                                   |
|  A: "Book a free discovery call"                                |
|                                                                   |
+------------------------------------------------------------------+
```

## Framework BAB (Before-After-Bridge)

```
+------------------------------------------------------------------+
|                                                                   |
|  FRAMEWORK BAB                                                   |
|  =============                                                   |
|                                                                   |
|  BEFORE: Leur situation actuelle (douloureuse)                   |
|  AFTER:  Leur vie apres (desirable)                              |
|  BRIDGE: Comment aller de Before a After (ton produit)           |
|                                                                   |
|  EXEMPLE:                                                        |
|  ========                                                        |
|                                                                   |
|  BEFORE: "Tu passes tes soirees a debugger du code legacy,      |
|           tu rates des deadlines, tes clients sont frustres."   |
|                                                                   |
|  AFTER:  "Tu livres en avance, tes clients te recommandent,     |
|           tu as le temps de travailler sur TES projets."        |
|                                                                   |
|  BRIDGE: "Dafnck Studio: CTO on-demand pour startups.           |
|           Je prends en charge le dev, tu te focus               |
|           sur la croissance."                                   |
|                                                                   |
+------------------------------------------------------------------+
```

# 4. Copy Qui Convertit

## Principes de Copywriting

```
+------------------------------------------------------------------+
|                                                                   |
|  LES 10 LOIS DU COPY QUI CONVERTIT                               |
|  ==================================                              |
|                                                                   |
|  1. PARLE A UNE PERSONNE                                         |
|     "Tu" pas "Les utilisateurs"                                 |
|     Comme si tu ecrivais a un ami                               |
|                                                                   |
|  2. BENEFICE D'ABORD                                             |
|     Ce qu'ils GAGNENT, pas ce que tu FAIS                       |
|     "Gagne 10h/semaine" > "Automatisation avancee"              |
|                                                                   |
|  3. SPECIFIQUE > GENERIQUE                                       |
|     "97% de satisfaction" > "Les clients adorent"               |
|     "En 4 semaines" > "Rapidement"                              |
|                                                                   |
|  4. SIMPLE > CLEVER                                              |
|     Comprehension immediate                                      |
|     Pas de jeux de mots forces                                  |
|                                                                   |
|  5. EMOTION + LOGIQUE                                            |
|     Emotion pour decider, logique pour justifier                |
|     Fais RESSENTIR puis RATIONALISER                            |
|                                                                   |
|  6. VOICE OF CUSTOMER                                            |
|     Utilise leurs mots exacts                                   |
|     Tire des interviews, reviews, support                       |
|                                                                   |
|  7. OBJECTION HANDLING                                           |
|     Anticipe les doutes                                         |
|     "Tu te demandes si..." "Voici pourquoi..."                  |
|                                                                   |
|  8. POWER WORDS                                                  |
|     Free, New, Now, You, Easy, Proven, Guaranteed               |
|     Instant, Secret, Discover                                   |
|                                                                   |
|  9. SCANNABLE                                                    |
|     Bullets, headers, bold                                      |
|     Personne ne lit tout                                        |
|                                                                   |
|  10. EDIT RUTHLESSLY                                             |
|      Premier draft / 2 = version finale                         |
|      Chaque mot doit meriter sa place                           |
|                                                                   |
+------------------------------------------------------------------+
```

## Structure de Paragraphe Persuasif

```
+------------------------------------------------------------------+
|                                                                   |
|  STRUCTURE: FEATURE -> BENEFIT -> PROOF                          |
|  =======================================                         |
|                                                                   |
|  FEATURE:  Ce que c'est                                          |
|  BENEFIT:  Ce que ca fait pour toi                               |
|  PROOF:    Pourquoi c'est credible                               |

|   ou tu progresses et ou tu bloques. Base sur l'analyse         |

+------------------------------------------------------------------+
```

# 6. CTAs Qui Font Cliquer

## Regles des CTAs

```
+------------------------------------------------------------------+
|                                                                   |
|  CTA RULES                                                       |
|  =========                                                       |
|                                                                   |
|  1. VERBE D'ACTION                                               |
|     |-- Start, Get, Try, Download, Join                         |
|     |-- PAS: Submit, Click here, Learn more                     |
|     +-- Action que IXLS font, pas que tu veux                   |
|                                                                   |
|  2. BENEFICE INCLUS                                              |
|     |-- "Start my free trial" > "Start"                         |
|     |-- "Get my growth guide" > "Download"                      |
|     +-- "Book my free call" > "Contact"                         |
|                                                                   |
|  3. URGENCE (si vraie)                                           |
|     |-- "Start free today"                                      |
|     |-- "Claim my spot"                                         |
|     +-- PAS de fausse urgence                                   |
|                                                                   |
|  4. FRICTION REDUCTION                                           |
|     |-- "No credit card required"                               |
|     |-- "Free forever"                                          |
|     +-- "Cancel anytime"                                        |
|                                                                   |
|  5. CONTRASTE VISUEL                                             |
|     |-- Couleur qui ressort                                     |
|     |-- Taille suffisante                                       |
|     +-- Espace autour                                           |
|                                                                   |
+------------------------------------------------------------------+
```
